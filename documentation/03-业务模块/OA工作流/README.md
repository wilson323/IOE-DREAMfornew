## 🎯 核心功能

### 流程定义管理
- 流程部署
- 流程定义查询
- 流程激活/禁用
- 流程删除

### 流程实例管理
- 流程启动
- 流程实例查询
- 流程挂起/激活
- 流程终止/撤销

### 任务管理
- 待办任务查询
- 已办任务查询
- 任务受理/取消
- 任务转办/委派
- 任务完成/驳回

### 流程监控
- 流程图可视化
- 流程历史记录
- 流程统计信息
- 用户工作量统计

## 🎨 UI设计特点

### Web端
- 现代化设计语言（Ant Design）
- 响应式布局
- 流畅的动画效果
- 直观的操作界面

### 移动端
- 简洁美观的界面
- 流畅的交互体验
- 支持手势操作
- 离线功能支持

## 📊 技术栈

### 后端
- Spring Boot 3.5.8
- MyBatis-Plus
- Sa-Token
- WebSocket

### 前端（Web）
- Vue 3
- Ant Design Vue 4
- TypeScript
- Axios

### 前端（移动端）
- Vue 3
- Vant 4
- TypeScript
- Axios
