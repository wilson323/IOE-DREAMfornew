# SmartAdmin 示例资料导航

> 版本: v1.0.0  
> 责任人: 架构组 × 测试组  
> 最后更新: 2025-11-12

## 目录结构
- `API_EXAMPLES.md`: 后端真实接口示例, 对应 `docs/COMMON_MODULES/api-contracts.md`。
- `TEST_SCRIPTS.md`: 后端、前端与端到端测试脚本说明。
- 所有字段命名遵循 `docs/DATA_DICTIONARY.md` 中的映射。

## 使用指南
1. 选择目标模块, 在 API 示例中复制请求与响应模板。
2. 测试脚本章节给出 PowerShell 命令, 请在 Windows 环境执行。
3. 如需新增示例, 同步更新索引与数据字典, 保证命名一致。
