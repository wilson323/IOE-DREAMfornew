## 🚀 一键启动指南

### 快速启动

**完整启动（推荐）**:
```powershell
# 启动所有服务（后端 + 前端 + 移动端）
.\scripts\start-all-complete.ps1
```

**分步启动**:
```powershell
# 仅启动后端微服务
.\scripts\start-all-complete.ps1 -BackendOnly

# 仅启动前端管理后台
.\scripts\start-all-complete.ps1 -FrontendOnly

# 仅启动移动端应用
.\scripts\start-all-complete.ps1 -MobileOnly

# 检查服务状态
.\scripts\start-all-complete.ps1 -CheckOnly
```

### 启动顺序

1. **后端微服务**（按依赖顺序）
   - 第1批次：网关服务（8080）
   - 第2批次：公共服务（8088）、设备通讯服务（8087）
   - 第3批次：OA服务（8089）、门禁服务（8090）、考勤服务（8091）、视频服务（8092）、消费服务（8094）、访客服务（8095）

2. **前端管理后台**（3000端口）
   - 自动检查依赖，如未安装则自动安装
   - 启动Vite开发服务器

3. **移动端应用**（8081端口，H5模式）
   - 自动检查依赖，如未安装则自动安装
   - 启动uni-app H5开发服务器

### 访问地址

启动完成后，可通过以下地址访问：

- **前端管理后台**: http://localhost:3000
- **移动端应用**: http://localhost:8081
- **API网关**: http://localhost:8080
