3. **距离适中**: 保持0.5-2米距离
4. **表情自然**: 保持正常表情

1. **预录入信息**: 提前录入高质量的人脸

### 消费使用技巧

## 🔧 常见问题快速解决

### 登录问题

#### 忘记密码
1. 点击"忘记密码"
2. 输入工号和验证码
3. 通过邮箱或手机验证
4. 设置新密码

#### 登录失败
1. 检查网络连接
2. 确认账号密码正确
3. 清除浏览器缓存
4. 联系IT部门

### 生物识别问题

#### 人脸识别失败
1. 调整光线和角度
2. 摘除遮挡物
3. 重新录入人脸
4. 使用其他方式

#### 指纹识别失败
1. 擦拭手指和传感器
2. 调整按压力度
3. 尝试其他手指
4. 重新录入指纹

### 支付问题

#### 支付失败
1. 检查账户余额
2. 检查网络连接
3. 尝试其他支付方式
4. 联系客服

#### 重复扣款
1. 保留消费凭证
2. 联系客服
3. 提供消费信息
4. 等待处理
