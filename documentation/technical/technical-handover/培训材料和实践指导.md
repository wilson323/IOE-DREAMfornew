### 2. 常见问题和解决方案

#### 2.1 环境问题

**问题1: JDK版本不兼容**
```bash
# 解决方案
# 1. 卸载旧版本JDK
sudo apt remove openjdk-8-jdk

# 2. 安装JDK 17
sudo apt install openjdk-17-jdk

# 3. 验证安装
java -version
javac -version

# 4. 配置环境变量
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export PATH=$JAVA_HOME/bin:$PATH
```

**问题2: Maven依赖下载失败**
```bash
# 解决方案
# 1. 配置国内镜像源
cp ~/.m2/settings.xml ~/.m2/settings.xml.backup

# 2. 编辑settings.xml添加阿里云镜像
<mirror>
  <id>aliyun-maven</id>
  <mirrorOf>central</mirrorOf>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>

# 3. 清理并重新下载
mvn clean
mvn dependency:resolve
```

#### 2.2 运行问题

**问题1: 服务启动失败**
```bash
# 解决方案
# 1. 查看启动日志
tail -f logs/application.log

# 2. 检查配置文件
cat application.yml

# 3. 检查端口占用
netstat -an | grep 8102

# 4. 检查依赖服务
ping mysql-host
telnet redis-host 6379
```

**问题2: 数据库连接失败**
```bash
# 解决方案
# 1. 检查数据库服务状态
systemctl status mysql

# 2. 检查网络连通性
telnet mysql-host 3306

# 3. 检查数据库用户权限
mysql -u root -p
SHOW GRANTS FOR 'root'@'%';

# 4. 更新连接配置
spring.datasource.url=jdbc:mysql://localhost:3306/ioedream?useSSL=false&serverTimezone=UTC
```
