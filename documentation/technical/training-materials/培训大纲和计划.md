# IOE-DREAM 智能管理系统培训大纲和计划

## 📖 培训概述

### 培训目标

IOE-DREAM智能管理系统培训旨在帮助不同角色的用户快速掌握系统功能，提高工作效率，确保系统能够充分发挥其价值。

#### 核心目标
- **功能掌握**: 全面了解系统各项功能模块
- **操作熟练**: 熟练掌握日常操作流程
- **问题解决**: 能够独立处理常见问题
- **安全意识**: 建立正确的安全使用意识
- **最佳实践**: 学习和掌握最佳使用实践

### 培训对象

| 角色 | 人数 | 培训重点 | 培训时长 |
|------|------|----------|----------|
| **普通员工** | 全体员工 | 基础功能使用 | 4小时 |
| **部门主管** | 各部门负责人 | 管理功能使用 | 6小时 |
| **HR人员** | HR部门员工 | 考勤管理功能 | 8小时 |
| **财务人员** | 财务部门员工 | 消费系统功能 | 6小时 |
| **安保人员** | 安保部门员工 | 门禁管理功能 | 8小时 |
| **系统管理员** | IT部门员工 | 系统运维管理 | 16小时 |
| **开发人员** | 开发团队 | 二次开发技能 | 20小时 |

### 培训方式

- **理论讲解**: 系统功能介绍和概念讲解
- **实际操作**: 现场演示和学员动手练习
- **案例分析**: 基于实际业务场景的案例
- **小组讨论**: 经验分享和问题讨论
- **在线学习**: 自学材料补充
- **考试认证**: 知识掌握程度评估

## 📚 培训课程体系

#### 课程名称: IOE-DREAM系统基础操作

**适合人群**: 全体员工

**培训目标**:
- 了解系统基本架构和功能

**模块2: 系统登录和设置 (45分钟)**

**模块3: 门禁管理基础 (45分钟)**

## 🧪 培训实验和练习

### 实验设计原则

1. **循序渐进**: 从简单到复杂，逐步提升难度
2. **实用导向**: 结合实际业务场景设计实验
3. **独立完成**: 学员能够独立完成操作
4. **错误处理**: 包含常见错误情况的处理
5. **文档支持**: 提供详细的实验指导文档

### 基础实验 (4个)

#### 实验1: 个人账户设置和生物识别录入
**实验目标**: 掌握账户管理和生物识别设置
**实验步骤**:
1. 登录系统并修改密码
2. 完善个人信息
3. 录入人脸识别信息
4. 录入指纹识别信息
5. 设置安全问题和答案

**考核标准**:
- 成功修改密码并能够使用新密码登录
- 个人信息完整且准确
- 生物识别信息录入成功且能够正常识别

#### 实验2: 门禁通行和访客预约
**实验目标**: 掌握门禁日常使用和访客管理
**实验步骤**:
1. 使用不同方式通过门禁
2. 为访客创建预约申请
3. 处理访客到现场登记
4. 查看通行记录和统计数据
5. 申请额外的门禁权限

**考核标准**:
- 能够使用至少2种方式通过门禁
- 成功创建访客预约并完成现场登记
- 能够查询和分析通行数据

#### 实验3: 考勤管理和请假流程
**实验目标**: 掌握考勤系统和请假申请
**实验步骤**:
1. 完成正常的考勤打卡
2. 查看个人考勤记录
3. 提交请假申请并跟踪状态
4. 处理考勤异常情况
5. 查看考勤统计报表

**考核标准**:
- 正确完成考勤打卡
- 成功提交请假申请
- 能够处理基本考勤异常

#### 实验4: 消费系统使用和账户管理
**实验目标**: 掌握消费系统的日常使用
**实验步骤**:
1. 查看账户余额和消费记录
2. 进行账户充值操作
3. 模拟餐厅消费流程

**考核标准**:
- 成功完成账户充值

**实际操作 (60分)**:

**实际操作 (60分)**:
