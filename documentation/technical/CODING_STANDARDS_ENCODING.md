# 编码标准化规范

## 编码标准
- **字符编码**: UTF-8
- **字节序标记(BOM)**: 不使用BOM
- **换行符**: LF (\\n)
- **缩进**: 4个空格
- **文件结尾**: 单个换行符

## 编码验证规则
1. 所有Java文件必须使用UTF-8编码
2. 不包含BOM标记
3. 中文字符正确显示
4. 禁止出现乱码字符

## 开发环境配置
- IDE编码设置: UTF-8
- Git配置: autocrlf=false
- Maven编码: UTF-8
- 终端编码: UTF-8

- 定期编码检查
- CI/CD编码验证
