## 🌟 系统概述

### 系统简介

IOE-DREAM智能管理系统是一套现代化的企业管理平台，集成了智能门禁、消费管理、考勤管理、视频监控等核心功能，为用户提供一体化、智能化的管理体验。

### 核心功能模块

| 功能模块 | 主要功能 | 适用人群 |
|---------|---------|----------|
| **智能门禁** | 人脸识别、指纹识别、访客管理、区域权限控制 | 全体员工、安保人员 |
| **消费系统** | 餐厅消费、商店购物、账户充值、消费记录 | 全体员工、财务人员 |
| **考勤管理** | 打卡签到、请假申请、加班管理、考勤统计 | 全体员工、HR人员 |
| **OA办公** | 审批流程、文档管理、内部通知、会议室预定 | 全体员工、行政人员 |
| **视频监控** | 实时监控、录像回放、智能分析、告警管理 | 安保人员、管理人员 |
| **系统监控** | 设备状态、系统性能、日志查看 | 技术人员、管理人员 |

### 系统特点

- **智能化**: 基于AI的生物识别技术，提升安全管理水平
- **一体化**: 多模块集成，统一管理平台
- **移动化**: 支持移动端访问，随时随地办公
- **安全性**: 多层次安全防护，保障数据安全
- **易用性**: 简洁直观的界面设计，操作简便

## 🚪 门禁管理

### 功能概述

智能门禁管理系统提供多种生物识别技术，支持人脸识别、指纹识别、虹膜识别等，实现安全便捷的门禁控制。

### 主要功能

#### 1. 人员通行
- **人脸识别通行**: 在门禁设备前站立，系统自动识别
- **指纹验证通行**: 在指纹设备上按压指纹进行验证
- **刷卡通行**: 使用员工卡或IC卡
- **二维码通行**: 使用手机App生成通行二维码

#### 2. 访客管理
- **访客预约**: 提前预约访客，生成临时通行凭证
- **现场登记**: 访客到现场登记信息，发放临时通行凭证
- **访客记录**: 查看访客通行记录和历史记录

#### 3. 区域权限管理
- **权限设置**: 为不同人员设置不同区域的访问权限
- **时间限制**: 设置访问时间范围和有效期
- **特殊授权**: 临时授权和紧急通道授权

### 操作指南

#### 人脸识别录入
1. 进入"个人中心" → "生物信息管理"
2. 点击"录入人脸"按钮
3. 按照提示调整姿势和表情
4. 系统自动采集和存储人脸信息
5. 录入完成后进行测试验证

#### 指纹信息录入
1. 进入"个人中心" → "生物信息管理"
2. 点击"录入指纹"按钮
3. 在指纹设备上多次按压同一手指
4. 系统提示录入成功
5. 进行测试验证

#### 申请门禁权限
1. 进入"门禁管理" → "权限申请"
2. 选择需要申请的区域和时间段
3. 填写申请原因
4. 提交申请等待审批
5. 审批通过后获得相应权限

#### 查看通行记录
1. 进入"门禁管理" → "通行记录"
2. 设置查询条件（日期、区域、人员等）
3. 点击"查询"按钮查看记录
4. 可导出记录报表

### 常见问题

#### 问题1: 人脸识别失败
**解决方案**:
- 确保光线充足
- 调整与摄像头的距离和角度
- 摘除帽子、口罩等遮挡物
- 如仍有问题，联系管理员重新录入

#### 问题2: 指纹验证不通过
**解决方案**:
- 确保手指干净干燥
- 正确按压指纹设备
- 尝试其他已录入的手指
- 如仍有问题，重新录入指纹

#### 问题3: 权限申请被拒绝
**解决方案**:
- 查看拒绝原因
- 补充必要信息
- 重新提交申请
- 联系相关负责人沟通

## 💳 消费系统

### 功能概述

消费管理系统提供便捷的员工消费服务，支持餐厅消费、商店购物、账户充值等功能，实现无现金化消费。

### 主要功能

#### 1. 餐厅消费
- **扫码点餐**: 通过手机App扫码点餐
- **刷脸支付**: 使用人脸识别进行支付
- **刷卡消费**: 使用员工卡消费
- **余额查询**: 实时查询账户余额

#### 2. 商店购物
- **商品浏览**: 查看商店商品信息
- **在线下单**: 通过App下单购买
- **自提服务**: 到店自提商品
- **配送服务**: 配送到指定地点

#### 3. 账户管理
- **账户充值**: 在线充值账户余额
- **交易记录**: 查看历史交易记录
- **消费统计**: 查看消费统计分析
- **退款申请**: 申请消费退款

### 操作指南

#### 账户充值
1. 进入"消费系统" → "账户管理"
2. 点击"账户充值"按钮
3. 选择充值金额或输入自定义金额
4. 选择支付方式（支付宝、微信、银行卡）
5. 确认支付信息并完成支付
6. 充值成功后余额立即更新

#### 餐厅消费
1. 到餐厅选择菜品
2. 到收银台进行结算
3. 选择支付方式：
   - **刷脸支付**: 在摄像头前进行人脸识别
   - **刷卡支付**: 使用员工卡
   - **扫码支付**: 使用手机App付款码
4. 确认支付信息
5. 支付成功后获取小票

#### 查看消费记录
1. 进入"消费系统" → "消费记录"
2. 设置查询条件：
   - 时间范围
   - 消费类型（餐厅、商店等）
   - 消费金额范围
3. 点击"查询"查看记录
4. 可导出消费记录报表

#### 申请退款
1. 进入"消费系统" → "退款申请"
2. 选择要退款的交易记录
3. 填写退款原因
4. 上传相关证明材料（如需要）
5. 提交申请
6. 等待管理员审核

### 消费规则

#### 每日消费限额
- **默认限额**: 每日200元
- **特殊申请**: 可申请提高限额
- **紧急情况**: 联系管理员临时调整

#### 消费时间
- **餐厅**: 早餐 7:00-9:00，午餐 11:00-14:00，晚餐 17:00-20:00
- **商店**: 9:00-21:00
- **线上商城**: 24小时开放

#### 支付方式
- **推荐方式**: 刷脸支付、刷脸+密码
- **备选方式**: 员工卡、手机扫码
- **特殊场景**: 现金、银行卡（需要管理员授权）

## ⏰ 考勤管理

### 功能概述

考勤管理系统提供智能考勤服务，支持多种打卡方式，自动生成考勤报表，简化考勤管理流程。

### 主要功能

#### 1. 打卡签到
- **人脸打卡**: 使用人脸识别技术打卡
- **指纹打卡**: 使用指纹识别技术打卡
- **定位打卡**: 基于GPS定位的移动打卡
- **二维码打卡**: 使用手机App生成打卡二维码

#### 2. 请假管理
- **请假申请**: 在线提交各类请假申请
- **审批流程**: 多级审批流程管理
- **请假记录**: 查看历史请假记录
- **假期余额**: 查看剩余假期余额

#### 3. 加班管理
- **加班申请**: 在线申请加班
- **加班记录**: 自动记录加班时长
- **调休管理**: 加班时长转换为调休
- **加班统计**: 查看加班统计报表

### 操作指南

#### 日常打卡
1. **上班打卡**:
   - 在公司门禁设备前进行人脸识别
   - 或打开手机App → 点击"上班打卡"
   - 系统显示打卡成功和时间
   - 生成打卡记录

2. **下班打卡**:
   - 在门禁设备前进行人脸识别
   - 或打开手机App → 点击"下班打卡"
   - 系统显示打卡成功和时间
   - 自动计算当日工作时长

#### 请假申请
1. 进入"考勤管理" → "请假申请"
2. 选择请假类型：
   - 事假
   - 病假
   - 年假
   - 调休
   - 婚假
   - 产假等
3. 选择请假开始和结束时间
4. 填写请假原因
5. 上传相关证明（如病假需要医疗证明）
6. 选择审批人
7. 提交申请等待审批

#### 加班申请
1. 进入"考勤管理" → "加班申请"
2. 选择加班日期和时间
3. 选择加班类型：
   - 工作日加班
   - 周末加班
   - 节假日加班
4. 填写加班事由
5. 选择是否需要调休
6. 提交申请等待审批

#### 查看考勤记录
1. 进入"考勤管理" → "考勤记录"
2. 设置查询条件：
   - 查询月份
   - 考勤状态（正常、迟到、早退、缺勤等）
3. 点击"查询"查看记录
4. 可导出考勤报表

### 考勤规则

#### 工作时间
- **标准工作时间**: 9:00-18:00（午休12:00-13:00）
- **弹性工作时间**: 可在8:30-9:30之间打卡，相应调整下班时间
- **加班时间**: 工作日超过18:00、周末、节假日

#### 考勤异常处理
- **迟到**: 超过规定打卡时间15分钟内
- **早退**: 未到规定下班时间提前离开
- **缺勤**: 未经请假未到岗
- **补卡**: 特殊情况下可申请补卡

#### 请假政策
- **年假**: 根据工龄和职级确定天数
- **病假**: 需提供医疗证明
- **事假**: 需提前申请，紧急情况可事后补申请
- **调休**: 加班时长可转换为调休时间

## 📋 OA办公系统

### 功能概述

OA办公系统提供完整的办公自动化功能，包括审批流程、文档管理、内部通知、会议室管理等，提高办公效率。

### 主要功能

#### 1. 审批流程
- **请假审批**: 请假申请的审批流程
- **报销审批**: 费用报销的审批流程
- **采购审批**: 物品采购的审批流程
- **用印审批**: 公司印章使用审批
- **出差审批**: 出差申请的审批流程

#### 2. 文档管理
- **文档上传**: 上传各类办公文档
- **文档分类**: 按部门、类型分类管理
- **权限控制**: 设置文档访问权限
- **版本管理**: 文档版本控制和历史记录
- **文档搜索**: 快速搜索所需文档

#### 3. 内部通知
- **通知发布**: 发布各类内部通知公告
- **通知分类**: 按类型和紧急程度分类
- **通知查看**: 查看历史通知记录
- **阅读确认**: 重要通知的阅读确认功能

#### 4. 会议室管理
- **会议室查看**: 查看所有会议室状态
- **会议室预定**: 在线预定会议室
- **预定记录**: 查看预定历史记录
- **使用统计**: 会议室使用情况统计

### 操作指南

#### 发起审批申请
1. 进入"OA办公" → "我的申请"
2. 点击"新建申请"按钮
3. 选择申请类型：
   - 请假申请
   - 报销申请
   - 采购申请
   - 用印申请
   - 出差申请
4. 填写申请表单信息
5. 上传相关附件
6. 选择审批流程和审批人
7. 提交申请

#### 处理审批任务
1. 进入"OA办公" → "待我审批"
2. 查看待审批的申请列表
3. 点击"查看详情"查看申请内容
4. 进行审批操作：
   - **同意**: 点击"同意"按钮，填写审批意见
   - **驳回**: 点击"驳回"按钮，填写驳回原因
   - **转审**: 将审批转给其他人
5. 提交审批结果

#### 文档上传和管理
1. 进入"OA办公" → "文档中心"
2. 点击"上传文档"按钮
3. 选择要上传的文件
4. 填写文档信息：
   - 文档标题
   - 文档分类
   - 关键词
   - 访问权限
5. 点击"上传"完成上传
6. 可在文档列表中查看和管理文档

#### 预定会议室
1. 进入"OA办公" → "会议室管理"
2. 查看会议室日历和状态
3. 选择合适的会议室和时间
4. 点击"预定"按钮
5. 填写会议信息：
   - 会议主题
   - 参会人数
   - 申请人
   - 联系方式
6. 确认预定信息
7. 提交预定申请

### 使用技巧

#### 审批流程优化
- **批量操作**: 支持批量审批相同类型申请
- **模板保存**: 保存常用申请表单为模板
- **快速回复**: 使用预设的快速回复内容

#### 文档管理技巧
- **标签分类**: 使用标签对文档进行细致分类
- **收藏功能**: 收藏常用文档方便快速访问
- **共享设置**: 设置文档共享给相关同事

### 功能概述

系统监控模块提供全面的系统监控功能，包括设备状态监控、性能监控、日志管理等，确保系统稳定运行。

### 主要功能

#### 1. 实时监控
- **设备状态**: 实时显示各设备运行状态
- **系统性能**: CPU、内存、磁盘、网络使用情况
- **服务状态**: 各微服务运行状态
- **告警信息**: 实时告警信息显示

#### 2. 视频监控
- **实时预览**: 实时查看各监控摄像头画面
- **录像回放**: 查看历史录像记录
- **云台控制**: 控制摄像头方向和变焦
- **智能分析**: 人流量统计、异常行为识别

#### 3. 日志管理
- **系统日志**: 查看系统运行日志
- **操作日志**: 查看用户操作记录
- **错误日志**: 查看系统错误信息
- **审计日志**: 查看安全审计记录

### 操作指南

#### 查看实时监控
1. 进入"系统监控" → "实时监控"
2. 查看各类监控指标：
   - **系统负载**: CPU使用率、内存使用率
   - **网络流量**: 上传下载速度
   - **存储空间**: 磁盘使用情况
   - **服务状态**: 各服务运行状态

#### 视频监控操作
1. 进入"系统监控" → "视频监控"
2. 选择要查看的摄像头
3. 点击"实时预览"查看实时画面

**A**:

3. 申请解锁账号
4. 重新设置密码

**A**:
1. 检查网络连接

3. 重启浏览器
4. 联系技术支持
