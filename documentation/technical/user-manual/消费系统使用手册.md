## 🌟 系统概述

### 消费系统简介

IOE-DREAM智能消费系统是集餐厅消费、商店购物、线上商城等功能于一体的综合消费管理平台，支持多种支付方式，为员工提供便捷、安全、智能的消费体验。

### 系统特点

- **多场景支持**: 餐厅、商店、线上商城全覆盖
- **智能支付**: 人脸识别、指纹、IC卡、手机支付
- **实时结算**: 交易实时处理，余额即时更新
- **数据分析**: 提供消费数据统计和分析
- **安全可靠**: 多重安全保障，防止消费欺诈

### 服务范围

#### 1. 餐厅服务
- **早餐时间**: 7:00-9:00
- **午餐时间**: 11:00-14:00
- **晚餐时间**: 17:00-20:00
- **夜宵时间**: 20:00-22:00（部分餐厅）

#### 2. 商店服务
- **便利店**: 8:00-22:00
- **文具店**: 9:00-18:00
- **生活超市**: 9:00-21:00
- **精品店**: 10:00-19:00

#### 3. 线上商城
- **商城时间**: 24小时开放
- **下单时间**: 全天可下单
- **配送时间**: 工作日9:00-17:00

## 💳 账户管理

### 账户信息

#### 账户状态
1. **正常状态**: 可正常使用所有消费功能
2. **冻结状态**: 暂时无法消费，需要解冻
3. **注销状态**: 账户已注销，无法使用
4. **欠费状态**: 需要充值后才能正常消费

#### 账户余额
- **实时余额**: 显示当前可用余额
- **冻结金额**: 预授权或争议中的金额
- **信用额度**: 可透支的金额（如适用）
- **累计消费**: 历史累计消费总额

### 账户设置

#### 支付密码设置
1. **设置路径**
   ```
   消费系统 → 账户设置 → 支付密码管理
   ```

2. **密码要求**
   - 长度：6位数字
   - 不能使用连续数字（如123456）
   - 不能使用重复数字（如111111）
   - 建议使用不规律的数字组合

3. **设置流程**
   - 输入新密码
   - 再次确认密码
   - 设置安全问题（用于找回密码）
   - 保存设置

#### 消费限额设置
1. **默认限额**
   - 单笔消费限额：100元
   - 每日消费限额：200元
   - 每月消费限额：3000元

2. **自定义限额**
   - 可申请调整消费限额
   - 需要主管审批
   - 特殊情况可临时调整

#### 消费提醒设置
1. **提醒类型**
   - 消费成功提醒
   - 余额不足提醒
   - 大额消费提醒
   - 异常消费提醒

2. **提醒方式**
   - 系统消息
   - 邮件通知
   - 短信通知
   - 手机APP推送

### 账户安全

#### 安全设置
1. **登录安全**
   - 设置登录密码
   - 启用短信验证
   - 绑定手机号码

2. **支付安全**
   - 设置支付密码
   - 启用消费提醒
   - 设置消费限额

3. **设备管理**
   - 查看已登录设备
   - 解绑异常设备
   - 定期更换设备密码

#### 安全检查
定期检查以下项目：
- 消费记录是否正常
- 账户余额是否正确
- 是否有异常登录记录
- 绑定信息是否正确

## 💰 支付方式

### 生物识别支付

#### 人脸识别支付
1. **支付流程**
   ```
   选择商品 → 确认金额 → 面向摄像头 → 人脸识别 → 输入支付密码 → 支付成功
   ```

2. **使用要点**
   - 确保光线充足
   - 摘除遮挡物
   - 保持正常表情
   - 距离摄像头0.5-2米

3. **安全措施**
   - 支付金额超过50元需要输入支付密码
   - 支持活体检测，防止照片欺骗
   - 异常情况会触发二次验证

#### 指纹识别支付
1. **支付流程**
   ```
   选择商品 → 确认金额 → 按压指纹 → 输入支付密码 → 支付成功
   ```

2. **注意事项**
   - 确保手指干燥清洁
   - 正确按压指纹传感器
   - 支付大额时需要密码确认

### 卡片支付

#### 员工卡支付
1. **支付流程**
   ```
   选择商品 → 确认金额 → 刷员工卡 → 输入密码（大额）→ 支付成功
   ```

2. **使用技巧**
   - 保持卡片清洁
   - 正确刷卡距离
   - 支付后及时拿回卡片

#### 手机支付
1. **二维码支付**
   ```
   打开手机APP → 生成付款码 → 扫码枪扫描 → 确认支付 → 支付成功
   ```

2. **NFC支付**（支持设备）
   ```
   手机靠近POS设备 → 确认支付 → 指纹/密码验证 → 支付成功
   ```

### 在线支付

#### 支付宝/微信支付
1. **支付流程**
   ```
   选择商品 → 确认金额 → 选择支付宝/微信 → 扫码支付 → 支付成功
   ```

2. **支付特点**
   - 实时到账
   - 支持大额支付
   - 有消费积分和优惠

#### 银行卡支付
1. **支付流程**
   ```
   选择商品 → 确认金额 → 选择银行卡支付 → 输入卡号信息 → 验证支付 → 支付成功
   ```

2. **安全保障**
   - 支持银联安全认证
   - 交易信息加密传输
   - 实时风控检测

## 🍽️ 餐厅消费

### 餐厅类型

#### 员工餐厅
1. **一楼餐厅**: 主要餐厅，提供家常菜、快餐
2. **二楼餐厅**: 特色餐厅，提供地方特色菜
3. **VIP餐厅**: 高级餐厅，需要预约
4. **咖啡厅**: 轻食餐厅，提供咖啡、简餐

#### 餐厅特色
- **一楼餐厅**: 经济实惠，菜品丰富
- **二楼餐厅**: 环境优雅，服务优质
- **咖啡厅**: 适合商务会谈
- **清真餐厅**: 满足特殊饮食需求

### 点餐方式

#### 现场点餐
1. **自助餐模式**
   - 按重量计价（元/100g）
   - 自助选择菜品
   - 在收银台称重结算

2. **套餐模式**
   - 固定价格套餐
   - 包含主食+配菜+汤
   - 可选择不同套餐

3. **单点模式**
   - 单独选择菜品
   - 按菜品单价计价
   - 可自由搭配

#### 手机APP点餐
1. **预点餐功能**
   ```
   打开APP → 餐厅预订 → 选择餐厅 → 浏览菜单 → 选择菜品 → 提交订单 → 确认时间 → 到店取餐
   ```

2. **外卖服务**
   - 支持办公室送餐
   - 起送金额：30元
   - 配送时间：30分钟内

3. **团体订餐**
   - 支持部门聚餐预订
   - 可预订包间
   - 提供定制菜单

### 支付流程

#### 结账流程
1. **选择结算方式**
   - 刷卡/刷脸支付
   - 手机扫码支付
   - 现金支付（部分餐厅）

2. **确认支付信息**
   - 核对消费金额
   - 确认账户余额
   - 选择优惠券（如有）

3. **完成支付**
   - 验证身份
   - 确认支付
   - 获取小票

#### 餐厅优惠
1. **日常优惠**
   - 会员折扣：95折
   - 满减优惠：满50减5元
   - 特价菜品：每日特价菜

2. **节日优惠**
   - 节日套餐优惠
   - 生日免单（当月生日）
   - 团体用餐优惠

## ⚠️ 异常处理

### 支付异常

#### 支付失败
1. **常见原因**:
   - 账户余额不足
   - 网络连接异常
   - 支付密码错误
   - 超过消费限额

2. **处理方法**:
   - 检查账户余额
   - 检查网络连接
   - 重试支付操作
   - 更换支付方式

#### 重复扣款
1. **识别方法**:
   - 查看消费记录
   - 检查账户余额
   - 对比消费小票

2. **处理流程**:
   ```
   发现重复扣款 → 联系客服 → 提供消费凭证 → 客服核实 → 退款处理
   ```

### 账户异常

#### 余额异常
1. **异常情况**:
   - 余额无故减少
   - 余额显示错误
   - 充值未到账

2. **处理步骤**:
   - 截图保留证据
   - 联系客服热线
   - 提供账户信息
   - 等待调查处理

#### 账户冻结
1. **冻结原因**:
   - 多次密码错误
   - 异常消费行为
   - 安全风险检测
   - 欠费未处理

2. **解冻流程**:
   - 联系客服或财务部
   - 提供身份验证
   - 说明冻结原因
   - 申请解冻处理

### 设备异常

#### POS机故障
1. **故障现象**:
   - 无法识别卡片
   - 网络连接失败
   - 打印机异常
   - 系统死机

2. **应急处理**:
   - 更换其他支付方式
   - 到其他收银台结账
   - 联系现场工作人员
   - 记录故障情况

#### 自助充值机故障
1. **处理方式**:
   - 选择其他充值方式
   - 联系财务部现场充值
   - 使用线上充值
   - 报告设备故障

## ❓ 常见问题

### 账户相关问题

#### Q1: 忘记支付密码怎么办？
**A**: 解决方法：
1. 使用"忘记密码"功能
2. 回答安全问题验证身份
3. 重新设置支付密码
4. 如无法自助解决，联系客服重置

#### Q2: 账户余额显示错误？
**A**: 处理步骤：
1. 刷新页面重新查看
2. 检查最近的消费记录
3. 联系客服查询详细账单
4. 如有错误，申请余额核对

#### Q3: 账户被冻结了怎么办？
**A**: 解冻方法：
1. 联系客服：400-123-4568
2. 提供身份验证信息
3. 了解冻结原因
4. 根据指导申请解冻

### 消费相关问题

#### Q4: 消费时余额不足怎么办？
**A**: 解决方案：
1. 现场充值（现金或刷卡）
2. 手机APP快速充值
3. 请同事代付并转账
4. 使用其他支付方式（支付宝、微信）

#### Q5: 消费记录有误怎么办？
**A**: 处理流程：
1. 保留消费小票
