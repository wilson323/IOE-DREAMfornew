## 🌟 系统概述

### 门禁系统简介

IOE-DREAM智能门禁管理系统采用先进的生物识别技术，集成人脸识别、指纹识别、虹膜识别等多种身份验证方式，为企业提供安全、便捷的门禁管理解决方案。

### 系统特点

- **多模态识别**: 支持人脸、指纹、虹膜、IC卡等多种识别方式
- **智能化管理**: AI驱动的智能分析和异常检测
- **移动化支持**: 支持手机App远程控制和查询
- **实时监控**: 实时监控各门禁点和人员通行情况
- **安全可靠**: 多重安全保障机制，防止非法闯入

### 门禁设备类型

#### 1. 人脸识别终端
- **设备型号**: SmartFace Pro
- **识别距离**: 0.5-3米
- **识别速度**: <1秒
- **适用场景**: 办公楼入口、重要区域

#### 2. 指纹识别终端
- **设备型号**: SmartFinger 2000
- **识别方式**: 半导体指纹传感器
- **存储容量**: 5000枚指纹
- **适用场景**: 机房、财务室等敏感区域

#### 3. 多功能门禁一体机
- **设备型号**: SmartGate All-in-One
- **支持方式**: 人脸+指纹+IC卡+密码
- **显示屏幕**: 8英寸触摸屏
- **适用场景**: 主入口、大厅

## 👤 生物识别设置

### 人脸识别录入

#### 录入前准备
1. **环境要求**:
   - 光线充足，避免强烈背光
   - 背景整洁，避免杂乱
   - 正常表情，不要遮挡面部

2. **个人准备**:
   - 摘除帽子、口罩、墨镜
   - 保持面部清洁，不要浓妆
   - 如果戴眼镜，建议摘除后录入

#### 录入流程
1. **进入录入页面**
   ```
   登录系统 → 个人中心 → 生物信息管理 → 录入人脸
   ```

2. **开始录入**
   - 点击"开始录入人脸"按钮
   - 系统启动摄像头

3. **姿势调整**
   - 正面朝向摄像头
   - 保持0.5-1.5米距离
   - 自然表情，不要过度夸张
   - 系统自动调整角度和光线

4. **信息采集**
   - 系统自动采集多角度人脸图像
   - 采集过程约10-15秒
   - 系统显示采集进度

5. **信息验证**
   - 采集完成后进行验证测试
   - 按照提示进行人脸识别测试
   - 验证成功后保存信息

#### 注意事项
- 每人最多可录入3个人脸档案
- 建议录入不同表情和角度的人脸
- 定期更新人脸信息（建议每6个月）

### 指纹识别录入

#### 录入前准备
1. **手指要求**:
   - 选择指纹清晰的手指
   - 避免手指有伤口或脱皮
   - 保持手指干燥清洁

2. **设备准备**:
   - 确保指纹传感器清洁
   - 检查设备连接正常

#### 录入流程
1. **进入录入页面**
   ```
   登录系统 → 个人中心 → 生物信息管理 → 录入指纹
   ```

2. **选择手指**
   - 系统显示手指选择界面
   - 建议选择食指或拇指
   - 每个手指最多录入2枚指纹

3. **开始录入**
   - 点击"开始录入"按钮
   - 将手指放在指纹传感器上

4. **多次采集**
   - 系统要求多次按压同一手指
   - 每次按压时稍微调整手指位置
   - 确保采集到完整的指纹信息
   - 通常需要5-8次按压

5. **质量检测**
   - 系统自动检测指纹质量
   - 如质量不合格，需要重新采集
   - 绿色指示表示采集成功

6. **验证测试**
   - 录入完成后进行验证
   - 用同一手指进行识别测试
   - 测试通过后保存信息

#### 录入技巧
- **按压力度**: 适中力度，不要太轻或太重
- **按压时间**: 保持2-3秒，不要移动手指
- **手指位置**: 每次尽量放在同一位置
- **重复尝试**: 如采集失败，可尝试其他手指

### 虹膜识别录入（高级功能）

#### 适用场景
- 高安全级别区域（如数据中心、研发实验室）
- 重要管理人员权限验证

#### 录入要求
1. **眼部健康**: 无眼部疾病或近期眼部手术
2. **隐形眼镜**: 建议取下隐形眼镜后录入
3. **化妆品**: 避免眼部浓妆

#### 录入流程
1. 预约虹膜录入时间
2. 由专业人员指导完成录入
3. 录入过程约5-10分钟
4. 录入完成后进行多次验证测试

## 🚪 门禁通行操作

### 日常通行操作

#### 人脸识别通行
1. **正常通行流程**
   ```
   接近门禁设备 → 设备自动激活 → 正面朝向摄像头 → 系统识别 → 门锁开启 → 通行
   ```

2. **操作步骤**
   - 距离设备0.5-3米
   - 自然走向设备
   - 保持正常行走速度
   - 不要突然停下或加速
   - 系统识别成功后会有提示音
   - 门锁开启后及时通行

3. **识别失败处理**
   - 退后一步重新尝试
   - 调整站立角度和距离
   - 确保光线充足
   - 如多次失败，使用其他验证方式

#### 指纹识别通行
1. **通行流程**
   ```
   到达设备前 → 按压指纹 → 系统验证 → 门锁开启 → 通行
   ```

2. **操作要点**
   - 将手指放在指纹传感器上
   - 保持手指稳定，不要移动
   - 按压2-3秒
   - 等待系统验证完成

#### IC卡/员工卡通行
1. **刷卡流程**
   ```
   取出员工卡 → 在读卡器上刷卡 → 系统验证 → 门锁开启 → 通行
   ```

2. **注意事项**
   - 保持卡片清洁干燥
   - 刷卡时保持卡片在读卡器上方2-5cm
   - 不要快速挥动卡片
   - 卡片损坏及时补办

#### 手机APP通行
1. **扫码通行**
   ```
   打开手机APP → 点击"开门" → 生成二维码 → 在设备上扫码 → 验证通过 → 门锁开启
   ```

2. **远程开门**
   - 在APP中选择对应门禁
   - 点击"远程开门"
   - 输入验证码或生物识别验证
   - 门锁远程开启

### 特殊情况通行

#### 手持物品通行
1. **最佳策略**:
   - 使用人脸识别（无需接触）
   - 避免使用指纹识别（需要放下物品）

2. **替代方案**:
   - 使用手机APP远程开门
   - 请同事协助开门
   - 联系安保人员协助

#### 多人同时通行
1. **原则**: 一人一验证，依次通行
2. **流程**:
   - 第一人验证通过并开门
   - 其他人跟随通行
   - 确保在门关闭时间内完成通行

#### 紧急通行
1. **紧急情况识别**:
   - 火灾、地震等灾害情况
   - 医疗急救情况
   - 其他突发事件

2. **处理方式**:
   - 使用紧急开门按钮（通常为红色）
   - 联系安保中心紧急开门
   - 使用应急钥匙（授权人员）

## 👥 访客管理

### 访客预约系统

#### 线上预约（推荐）
1. **预约流程**
   ```
   登录系统 → 访客管理 → 预约访客 → 填写访客信息 → 选择访问时间 → 提交申请 → 等待审批
   ```

2. **预约信息**
   - 访客姓名、身份证号
   - 联系电话
   - 访问目的和事由
   - 访问时间和日期
   - 访问区域和楼层
   - 接待人信息

3. **审批流程**
   - 系统自动分配审批人
   - 审批人收到邮件/短信通知
   - 在线审批或驳回
   - 审批结果通知申请人

#### 现场登记
1. **登记流程**
   ```
   到达前台 → 出示身份证件 → 填写访问信息 → 获得临时通行凭证 → 接待人签字确认
   ```

2. **所需材料**
   - 身份证原件
   - 工作证（如有）
   - 预约信息（如有）

3. **临时凭证类型**
   - 临时IC卡
   - 二维码通行证
   - 人脸临时权限（需要现场录入）

### 访客通行管理

#### 临时通行凭证
1. **IC卡临时卡**
   - 有效期：当天或指定时间
   - 使用权限：指定区域
   - 使用次数：一次或多次

2. **二维码通行证**
   - 有效期：通常当天
   - 显示方式：手机或打印件
   - 验证方式：设备扫码

3. **人脸临时权限**
   - 需要在现场临时录入人脸
   - 适用于重要访客
   - 有效期可灵活设置

#### 访客陪同制度
1. **基本原则**:
   - 重要区域必须有人陪同
   - 陪同人员全程负责访客安全
   - 陪同人员需全程在场

2. **陪同人员责任**:
   - 引导访客到指定区域
   - 确保访客不越权访问
   - 处理访客的紧急需求
   - 访客离开时确认交接

### 访客记录管理

#### 访客信息查询
1. **查询路径**
   ```
   门禁管理 → 访客管理 → 访客记录查询
   ```

2. **查询条件**
   - 访客姓名
   - 访问时间范围
   - 访问目的
   - 接待人

#### 访客统计报表
1. **统计维度**
   - 日/周/月访客数量

   - 访问区域分布

2. **报表导出**

   - 自定义时间范围

## 🚨 应急处理

### 紧急开门

#### 紧急情况识别
1. **自然灾害**:
   - 火灾、地震、洪水
   - 极端天气
   - 停电等设备故障

2. **安全威胁**:
   - 发现可疑人员
   - 设备被破坏
   - 其他安全威胁

3. **医疗急救**:
   - 突发疾病
   - 意外伤害
   - 需要紧急医疗救助

#### 紧急开门方式
1. **紧急按钮**:
   - 位置：门禁设备侧面或下方
   - 标识：红色按钮，标有"紧急"字样
   - 操作：按下按钮3秒
   - 效果：立即解除门锁

2. **远程紧急开门**:
   - 联系安保中心：400-123-4567
   - 提供身份信息和紧急情况描述
   - 安保人员远程开启门锁

3. **机械钥匙**:
   - 适用人员：安保人员、物业管理人员
   - 操作：使用专用机械钥匙开门
   - 注意：仅在紧急情况下使用

### 设备故障处理

#### 常见故障类型
1. **识别失败**:
   - 人脸识别不工作
   - 指纹识别不工作
   - IC卡读取失败

2. **设备无响应**:
   - 屏幕不亮
   - 按键无反应
   - 网络连接中断

3. **门锁故障**:
   - 门锁无法开启
   - 门锁无法关闭
   - 异常声音

#### 故障处理步骤
1. **初步判断**:
   - 检查设备电源
   - 检查网络连接
   - 查看错误提示

2. **尝试解决**:
   - 重启设备（断电30秒后重新通电）
   - 检查网络线路
   - 尝试其他验证方式

3. **报告故障**:
   ```
   故障报告内容：
   - 故障设备位置和编号
   - 故障现象描述
   - 发现时间
   - 联系人信息
   ```

4. **联系技术支持**:
   - 内部技术支持：内线8888
   - 供应商技术支持：400-567-8901
   - 紧急维修：186-XXXX-XXXX

### 丢失物品处理

#### IC卡/员工卡丢失
1. **立即挂失**:
   - 电话挂失：400-123-4567
   - 线上挂失：登录系统挂失
   - 现场挂失：到安保部挂失

2. **补办流程**:
   - 填写补卡申请表

## ❓ 常见问题

### 识别相关问题

#### Q1: 人脸识别经常失败怎么办？
**A**: 解决步骤：
1. 检查环境光线是否充足
2. 调整与摄像头的距离（0.5-3米）
3. 摘除帽子、口罩、眼镜等遮挡物
4. 尝试不同的面部角度
5. 如仍无法解决，重新录入人脸信息
6. 联系技术支持：400-123-4567

#### Q2: 指纹识别不灵敏？
**A**: 原因和解决方案：
1. **手指湿润**: 擦干手指再试
2. **手指干燥**: 用湿纸巾轻擦手指
3. **指纹磨损**: 尝试其他手指
4. **传感器脏污**: 联系清洁设备
5. **重新录入**: 考虑重新录入指纹

#### Q3: IC卡无法使用？
**A**: 检查和解决：
1. 检查卡片是否损坏
2. 确认卡片未过期
3. 检查账户是否正常
4. 确认是否有相关权限
5. 如卡片损坏，申请补卡

### 权限相关问题

#### Q4: 某个区域无法进入？
**A**: 处理步骤：
1. 确认是否拥有该区域权限
2. 检查权限是否在有效期内
3. 确认是否在允许的时间范围内
4. 如需要权限，申请相应区域访问权限
5. 联系直属主管或安保部门

#### Q5: 权限申请被拒绝了？
**A**: 处理方法：
1. 查看拒绝原因
2. 补充必要的申请材料
3. 与审批人沟通申请需求
4. 修改申请内容后重新提交
5. 如有疑问，联系安保部门咨询

### 设备相关问题

#### Q6: 门禁设备没有反应？
**A**: 检查步骤：
1. 检查设备是否通电（指示灯）
2. 检查网络连接是否正常
3. 尝试其他验证方式
4. 联系技术支持报告故障
5. 使用其他入口或联系安保人员

#### Q7: 门锁无法开启？
**A**: 解决方案：
1. 确认身份验证已通过
2. 等待系统处理（通常2-3秒）
3. 如仍无法开启，尝试重新验证
4. 使用紧急开门方式（如适用）
5. 联系安保人员协助

### 访客相关问题

#### Q8: 访客无法进入？
**A**: 处理流程：
1. 确认访客已成功预约
2. 检查预约是否已通过审批
3. 确认访客通行凭证是否有效
4. 如未预约，协助现场登记
5. 联系前台或安保部门协助

#### Q9: 访客通行凭证丢失？
**A**: 处理方法：
1. 立即联系安保部门
2. 临时冻结该通行凭证
3. 为访客重新发放凭证
4. 监控原凭证使用情况
5. 访客离开时确认凭证回收

### 系统使用问题

#### Q10: 忘记门禁密码？
**A**: 解决方法：
1. 使用其他验证方式（人脸、指纹、IC卡）
2. 联系IT部门重置密码
3. 在系统内使用"忘记密码"功能
4. 到安保部门现场重置

#### Q11: 手机APP无法开门？
**A**: 检查步骤：
1. 检查手机网络连接
2. 确认APP版本为最新
3. 检查账户是否正常
4. 重新登录APP
5. 如仍有问题，联系技术支持
