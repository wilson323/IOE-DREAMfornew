# SmartAdmin 测试覆盖率报告模板

> 版本: v1.0.0  
> 责任人: 测试保障组  
> 最后更新: 2025-11-12

## 1. 覆盖要求
- 单元测试覆盖率 ≥ 80%, 核心业务 ≥ 90%;
- 前端端到端测试覆盖关键流程 (登录、菜单、核心操作);
- 所有变更需包含正向与异常用例。

## 2. 收集命令示例
```powershell
# 后端
mvn test -DskipITs
mvn jacoco:report

# 前端
npm run test:unit -- --coverage
npm run test:e2e
```

## 4. 提交流程
1. 测试执行完毕后, 填写本模板并存档 `docs/testing/reports/`;
2. 若覆盖率低于标准, 必须附整改计划;
3. 将报告与 PR 关联, 并通知代码审查者。

## 5. 工具说明
- Jacoco 0.8.11, Jest 29+, Playwright 1.49+;
- 覆盖率数据需导入质量平台供审计; 参考 `SmartAdmin规范体系_v4/质量度量体系`。

## 6. 变更管理
- 模板更新需同步 `CHANGELOG.md`;
- 遇到新指标需求, 先提交质量评审。
