## 📝 标准转换指令模板

### 模板A：直接创建文件（推荐）⭐

**复制下面的指令，把HTML代码粘贴进去即可：**

```
请严格按照《SmartAdmin AI 二次开发指令文档》的规范，
将下面的HTML代码转换成符合项目规范的Vue 3代码，
并使用 write 工具直接创建文件到项目中。

【在这里粘贴你的HTML代码】

转换要求：

1. 自动分析并提取：
   - 页面类型（列表页/表单页/Dashboard等）
   - 所有字段及其类型、验证规则
   - 表格列
   - 搜索条件
   - 操作按钮和权限

2. 使用 write 工具直接创建以下文件：
   - src/api/业务模块/xxx-api.js（API接口文件）
   - src/constants/业务模块/xxx-const.js（常量定义文件）
   - src/views/业务模块/xxx/index.vue（列表页组件）
   - src/views/业务模块/xxx/components/xxx-form-modal/index.vue（表单弹窗组件）

3. 必须遵循的规范：
   ✅ 文件名使用 kebab-case
   ✅ 组件名使用 PascalCase
   ✅ API/变量使用 camelCase
   ✅ 常量使用 UPPER_SNAKE_CASE
   ✅ 路径使用 /@/ 别名
   ✅ 所有文件包含标准文件头注释
   ✅ 使用 setup script 语法
   ✅ 错误处理使用 smartSentry.captureError
   ✅ 权限控制使用 v-privilege 指令

4. 技术栈要求：
   - Vue 3 + Composition API
   - Ant Design Vue 4
   - 使用项目已有的组件（SmartEnumSelect、FileUpload等）
   - 图表使用 ECharts

5. 代码组织顺序：
   - 导入依赖（Vue API → 第三方库 → 本地模块）
   - Props/Emits定义
   - 响应式数据
   - 计算属性
   - 方法定义
   - 生命周期

请开始转换，直接创建文件到项目中。
```

## 🎓 智能识别能力

**AI可以自动识别：**

| HTML元素 | AI自动识别为 | 生成的Vue组件 |
|---------|------------|-------------|
| `<input type="text">` | 文本字段 | `<a-input>` |
| `<input type="number">` | 数字字段 | `<a-input-number>` |
| `<select>` | 枚举选择 | `<a-select>` 或 `<SmartEnumSelect>` |
| `<textarea>` | 文本域 | `<a-textarea>` |
| `<input type="file">` | 文件上传 | `<FileUpload>` |
| `<table>` | 数据表格 | `<a-table>` |
| `<form>` | 表单 | `<a-form>` |
| `required` | 必填 | 验证规则 `required: true` |
| `maxlength="50"` | 长度限制 | 验证规则 `max: 50` |
| 富文本编辑器 | 富文本 | `<WangEditor>` |

**AI可以自动推断：**

- ✅ 字段是否必填
- ✅ 字段验证规则
- ✅ 枚举值定义
- ✅ 权限命名
- ✅ API接口命名
- ✅ 模块分类（business/support/system）
