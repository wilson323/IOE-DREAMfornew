## 规范体系概述

IOE-DREAM开发规范体系是基于SmartAdmin规范体系v4精华内容，结合repowiki文档标准构建的统一、权威、专业的企业级开发规范。采用五层治理架构设计，为IOE-DREAM智慧园区一卡通管理平台项目提供完整的技术规范指导。

## 📋 规范使用指南

### 不同角色的阅读路径

**架构师**：
1. 架构设计规范 → 2. 数据库设计规范 → 3. 架构最佳实践 → 4. 性能优化案例

**后端开发**：
1. Java编码标准 → 2. API设计规范 → 3. Controller/Service模板 → 4. 单元测试指南

**前端开发**：
1. Vue3开发规范 → 2. API设计规范 → 3. 组件模板 → 4. 性能优化案例

**测试工程师**：
1. 单元测试指南 → 2. 系统安全规范 → 3. 常见问题解决 → 4. 代码质量标准

**AI辅助开发**：
1. AI开发指令集 → 2. AI代码审查标准 → 3. 对应核心规范 → 4. 代码模板
