## 🔧 提供的清理工具

### 清理脚本文件
1. **Linux/macOS**: `scripts/cleanup-project.sh`
2. **Windows**: `scripts/cleanup-project.ps1`

### 脚本功能特性
- ✅ **安全备份**: 自动创建Git备份分支
- ✅ **预演模式**: 支持--dry-run预览清理效果
- ✅ **分阶段执行**: 按优先级分阶段清理
- ✅ **详细报告**: 生成完整的清理执行报告
- ✅ **错误处理**: 完善的错误处理和回滚机制

### 清理脚本使用方法

#### Windows环境执行
```powershell
# 预演模式（推荐先执行）
.\scripts\cleanup-project.ps1 -DryRun

# 正式执行
.\scripts\cleanup-project.ps1

# 强制执行（跳过确认）
.\scripts\cleanup-project.ps1 -Force
```

#### Linux/macOS环境执行
```bash
# 给脚本执行权限
chmod +x scripts/cleanup-project.sh

# 预演模式
./scripts/cleanup-project.sh --dry-run

# 正式执行
./scripts/cleanup-project.sh
```
