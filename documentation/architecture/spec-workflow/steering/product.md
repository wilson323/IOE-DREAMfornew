# IOE-DREAM 智慧园区综合管理平台产品概览

## 产品定位

IOE-DREAM是一个基于SmartAdmin v3框架的智慧园区综合管理平台，为产业园区、校园、商业综合体等场景提供一站式智能管理解决方案。平台整合了门禁管理、消费管理、访客管理、考勤管理、视频监控等核心业务模块，通过统一的数字化平台提升园区管理效率和用户体验。

## 目标用户

### 主要用户群体

1. **园区管理员**
   - 痛点：多个独立系统管理复杂，数据孤岛严重
   - 需求：统一管理平台，实时监控，数据整合分析

2. **企业/商户用户**
   - 痛点：访客预约流程繁琐，消费对账复杂
   - 需求：便捷的访客管理，自动化对账结算

3. **个人用户**
   - 痛点：多卡多证，通行体验不便捷
   - 需求：一卡通通行，移动端自助服务

4. **运维技术人员**
   - 痛点：设备维护困难，故障定位慢
   - 需求：设备统一管理，远程监控诊断

## 核心功能

### 1. 智能门禁管理系统
- **人脸识别通行**：无感快速通行，支持多模态生物识别
- **区域权限管理**：精细化的空间权限控制，支持时间窗口管理
- **访客预约审批**：线上预约流程，临时权限发放
- **设备统一管理**：多品牌门禁设备接入，状态实时监控

### 2. 综合消费管理系统
- **多场景支付**：餐厅、超市、停车等场景统一支付
- **账户管理**：个人账户、企业账户、补助账户管理
- **对账结算**：自动化对账，多维度财务报表
- **移动支付**：支持二维码、NFC等多种支付方式

### 3. 智能考勤管理系统
- **多模式打卡**：人脸识别、IC卡、移动定位等多种打卡方式
- **排班管理**：灵活的排班规则，自动班次计算
- **异常处理**：请假、加班、外勤等异常流程自动化
- **统计分析**：考勤数据统计，薪资计算辅助

### 4. 视频监控管理系统
- **实时监控**：多画面实时查看，PTZ控制
- **智能分析**：行为识别、异常检测、人流统计
- **录像管理**：云端存储，快速检索回放
- **报警联动**：与门禁、报警系统联动
