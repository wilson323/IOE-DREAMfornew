# 行为分析模块 - 用户故事

## US-AI-001: 人脸抓拍

**作为** 安保人员(Level 3)  
**我希望** 系统自动抓拍经过的人脸  
**以便** 记录进出人员信息

### 验收标准
- [ ] 人脸检测准确率 ≥ 95%
- [ ] 抓拍延迟 < 200ms
- [ ] 自动过滤重复抓拍
- [ ] 保存清晰人脸图片

### 前置条件
- 设备支持人脸检测
- 已启用人脸抓拍规则

### 操作步骤
1. 系统自动检测人脸
2. 抓拍最佳质量帧
3. 保存抓拍记录
4. 可查看抓拍历史
