# 告警管理模块 - 用户故事

## US-ALM-001: 告警接收

**作为** 安保人员(Level 3)  
**我希望** 实时接收各类告警通知  
**以便** 及时响应安全事件

### 验收标准
- [ ] 告警推送延迟 < 1秒
- [ ] 支持WebSocket实时推送
- [ ] 高优先级告警弹窗提醒
- [ ] 支持声音/震动提醒

### 前置条件
- 用户已登录且权限 >= Level 2
- 已订阅告警通知

### 操作步骤
1. 系统检测到异常事件
2. 生成告警并入库
3. 推送给订阅用户
4. 用户收到告警通知

---
