# 解码上墙模块 - 需求规格说明

## 1. 功能需求

### 1.1 解码器管理
| 需求ID | 功能描述 | 优先级 |
|--------|----------|--------|
| DM-001 | 解码器设备添加/编辑/删除 | P0 |
| DM-002 | 解码器状态监控 | P0 |
| DM-003 | 解码器通道管理 | P0 |
| DM-004 | 解码器参数配置 | P1 |

### 1.2 电视墙管理
| 需求ID | 功能描述 | 优先级 |
|--------|----------|--------|
| TW-001 | 电视墙布局配置 | P0 |
| TW-002 | 屏幕位置拖拽调整 | P0 |
| TW-003 | 分屏模式设置(1/4/9/16) | P0 |
| TW-004 | 解码器-屏幕绑定 | P0 |

### 1.3 上墙操作
| 需求ID | 功能描述 | 优先级 |
|--------|----------|--------|
| WO-001 | 拖拽视频到窗口上墙 | P0 |
| WO-002 | 批量上墙 | P1 |
| WO-003 | 窗口切换/关闭 | P0 |
| WO-004 | 音频切换 | P2 |

### 1.4 预案管理
| 需求ID | 功能描述 | 优先级 |
|--------|----------|--------|
| PM-001 | 保存当前布局为预案 | P0 |
| PM-002 | 一键调用预案 | P0 |
| PM-003 | 预案定时切换 | P1 |
| PM-004 | 告警联动预案 | P1 |

### 1.5 轮巡功能
| 需求ID | 功能描述 | 优先级 |
|--------|----------|--------|
| PT-001 | 轮巡计划配置 | P1 |
| PT-002 | 轮巡间隔设置 | P1 |
| PT-003 | 轮巡启停控制 | P1 |

## 2. 非功能需求

### 2.1 性能要求
| 指标 | 要求 |
|------|------|
| 上墙延迟 | ≤ 2秒 |
| 预案切换时间 | ≤ 5秒(全量) |
| 画面流畅度 | ≥ 25fps |
| 最大并发上墙 | ≥ 64路 |

### 2.2 可靠性要求
- 解码器断线自动重连
- 预案切换失败自动回滚
- 上墙状态实时同步

## 3. 验收标准

- [ ] 电视墙布局配置正常
- [ ] 拖拽上墙延迟≤2秒
- [ ] 预案一键切换有效
- [ ] 轮巡功能正常运行
- [ ] 画面流畅度≥25fps
