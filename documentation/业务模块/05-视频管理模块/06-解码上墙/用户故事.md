# 解码上墙模块 - 用户故事

## US-WALL-001: 手动上墙

**作为** 安保人员(Level 3)  
**我希望** 将视频拖拽到电视墙窗口  
**以便** 在大屏上查看监控画面

### 验收标准
- [ ] 拖拽操作响应流畅
- [ ] 上墙延迟 < 2秒
- [ ] 显示上墙状态
- [ ] 支持替换已有画面

### 前置条件
- 用户已登录且权限 >= Level 3
- 解码器在线

### 操作步骤
1. 在设备树选择摄像机
2. 拖拽到目标窗口
3. 系统执行上墙
4. 电视墙显示画面
