# 访客管理模块

## 模块概述

访客管理模块是IOE-DREAM智能园区管理系统的核心业务模块之一，提供完整的访客接待全流程管理能力，包括访客预约、身份验证、登记管理、通行控制、物流管理和统计分析等功能。

## 技术架构

- **微服务**: Visitor Service (8095)
- **技术栈**: Spring Boot 3.5.8 + Java 17
- **数据库**: MySQL 8.0 + Druid连接池
- **缓存**: Redis + Caffeine多级缓存
- **认证**: Sa-Token
