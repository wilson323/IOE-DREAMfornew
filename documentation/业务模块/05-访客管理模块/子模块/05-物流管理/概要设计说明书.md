# 物流管理子模块 - 概要设计说明书

> **版本**: v1.0.0  
> **更新日期**: 2025-01-30

## 1. 系统架构

### 1.1 模块定位

物流管理负责访客携带物品的全流程管理，确保物品安全。

### 1.2 架构图

```
┌─────────────────────────────────────────────────┐
│                  物流管理模块                     │
├─────────────────────────────────────────────────┤
│  Controller层                                    │
│  └── LogisticsController (物流管理)              │
├─────────────────────────────────────────────────┤
│  Service层                                       │
│  └── LogisticsService                           │
├─────────────────────────────────────────────────┤
│  Manager层                                       │
│  ├── LogisticsManager                           │
│  └── StorageLocationManager                     │
├─────────────────────────────────────────────────┤
│  DAO层                                          │
│  └── LogisticsDao                               │
└─────────────────────────────────────────────────┘
```

## 2. 模块设计

### 2.1 核心模块

| 模块 | 职责 |
|------|------|
| 寄存管理 | 物品登记、照片拍摄、位置分配 |
| 领取管理 | 身份核验、物品归还 |
| 存放位置 | 位置管理、容量监控 |
| 超期处理 | 超期提醒、转交处理 |

## 3. 接口概览

| 接口 | 方法 | 说明 |
|------|------|------|
| /logistics/deposit | POST | 物品寄存 |
| /logistics/pickup | POST | 物品领取 |
| /logistics/list | GET | 寄存列表 |
| /logistics/overdue | GET | 超期物品 |
