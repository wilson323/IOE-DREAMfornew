# 身份验证子模块 - 验收标准

## 1. 功能验收

### 1.1 证件识别
- [ ] 支持二代身份证识别
- [ ] 识别准确率≥98%
- [ ] 自动提取姓名、身份证号、有效期
- [ ] 检测身份证是否过期

### 1.2 人脸验证
- [ ] 支持活体检测
- [ ] 人脸比对准确率≥99%
- [ ] 支持多角度人脸
- [ ] 相似度分数可配置

### 1.3 黑名单管理
- [ ] 支持添加/移除黑名单
- [ ] 支持设置黑名单有效期
- [ ] 黑名单实时生效
- [ ] 支持批量导入

## 2. 性能验收

- [ ] OCR识别≤2秒
- [ ] 人脸比对≤1秒
- [ ] 黑名单查询≤100ms

## 3. 安全验收

- [ ] 敏感数据加密存储
- [ ] 人脸数据定期清理
- [ ] 验证日志完整记录
