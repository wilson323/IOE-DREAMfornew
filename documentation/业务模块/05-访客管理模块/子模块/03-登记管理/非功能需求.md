# 登记管理子模块 - 非功能需求

## 1. 性能需求

| 指标 | 要求 |
|------|------|
| 签到响应时间 | ≤1秒（含人脸识别） |
| 签出响应时间 | ≤500ms |
| 在场查询响应 | ≤200ms |
| 并发签到处理 | ≥50TPS |

## 2. 可靠性需求

| 指标 | 要求 |
|------|------|
| 系统可用性 | ≥99.9% |
| 签到成功率 | ≥99.5% |
| 数据一致性 | 强一致性 |
| 离线支持 | 支持离线签到 |

## 3. 安全性需求

| 需求 | 说明 |
|------|------|
| 人脸数据保护 | 加密存储，定期清理 |
| 访客卡安全 | 一卡一用，实时状态 |
| 权限验证 | 区域权限实时校验 |
| 操作审计 | 全程操作记录 |

## 4. 易用性需求

| 需求 | 说明 |
|------|------|
| 自助签到 | 支持自助终端 |
| 快速签到 | 扫码/人脸秒级签到 |
| 状态展示 | 实时在场状态 |
| 异常提示 | 清晰的异常引导 |
