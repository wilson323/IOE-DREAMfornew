# 登记管理子模块 - 验收标准

## 1. 功能验收

### 1.1 签到功能
- [ ] 预约访客可扫码签到
- [ ] 支持人脸识别签到
- [ ] 临时访客可现场登记
- [ ] 签到后自动发放访客卡
- [ ] 签到后生成动态二维码

### 1.2 签出功能
- [ ] 访客可主动签出
- [ ] 支持前台代签出
- [ ] 超时自动签出
- [ ] 签出时回收访客卡

### 1.3 在场监控
- [ ] 实时显示在场访客数
- [ ] 支持按区域查看分布
- [ ] 超时访客自动预警
- [ ] 支持导出在场列表

## 2. 性能验收

- [ ] 签到响应时间≤1秒
- [ ] 签出响应时间≤500ms
- [ ] 支持50+并发签到

## 3. 安全验收

- [ ] 人脸数据加密存储
- [ ] 访客卡状态实时同步
- [ ] 操作日志完整记录
