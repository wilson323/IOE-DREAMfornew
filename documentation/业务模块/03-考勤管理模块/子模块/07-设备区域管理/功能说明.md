# 设备区域管理 - 功能说明

> **版本**: v1.0.0  
> **微服务**: ioedream-attendance-service (8091)  
> **创建日期**: 2025-12-17

---

## 📋 功能概述

设备区域管理模块负责管理考勤设备和考勤区域的配置，包括设备绑定、区域划分、设备与区域的关联关系等。

---

## 🎯 核心功能

### 1. 考勤设备管理

- **设备登记**: 登记考勤机、门禁设备等
- **设备配置**: 配置设备参数和通信方式
- **设备状态**: 监控设备在线状态
- **设备维护**: 设备维护和故障处理

### 2. 考勤区域管理

- **区域划分**: 按楼层、区域划分考勤区域
- **区域层级**: 支持多级区域层级结构
- **区域权限**: 区域访问权限控制
- **区域地图**: 可视化区域地图展示

### 3. 设备区域关联

- **设备绑定**: 设备绑定到指定区域
- **区域设备**: 查看区域内的设备列表
- **关联规则**: 配置设备与区域的关联规则

### 4. 考勤点配置

- **考勤点定义**: 定义GPS考勤点
- **有效范围**: 设置考勤点有效半径
- **考勤点分组**: 考勤点分组管理

## 🔧 配置参数

| 参数 | 类型 | 说明 |
|------|------|------|
| device_code | String | 设备编码 |
| device_name | String | 设备名称 |
| device_type | String | 设备类型 |
| device_ip | String | 设备IP地址 |
| device_port | Integer | 设备端口 |
| area_id | Long | 所属区域ID |
| status | Integer | 设备状态 |
