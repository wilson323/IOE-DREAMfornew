# 班次时间管理 - 功能说明

> **版本**: v1.0.0  
> **微服务**: ioedream-attendance-service (8091)  
> **创建日期**: 2025-12-17

---

## 📋 功能概述

班次时间管理模块负责企业考勤班次的定义和配置，包括工作时间、休息时间、打卡时间窗口等核心参数的管理。

---

## 🎯 核心功能

### 1. 班次定义
- **功能**: 创建和管理班次基本信息
- **包含**: 班次名称、班次代码、班次类型
- **班次类型**: 早班、中班、晚班、弹性班、通宵班

### 2. 工作时间配置
- **上班时间**: 工作开始时间
- **下班时间**: 工作结束时间
- **工作时长**: 自动计算或手动设置

### 3. 休息时间配置
- **休息开始**: 休息时间开始
- **休息结束**: 休息时间结束
- **多段休息**: 支持配置多个休息时段

### 4. 打卡时间窗口
- **提前打卡**: 允许提前多少分钟打卡
- **延后打卡**: 允许延后多少分钟打卡
- **有效范围**: 打卡有效时间范围

### 5. 弹性班次
- **弹性时间**: 弹性上下班时间范围
- **核心时段**: 必须在岗的核心工作时间
- **工时要求**: 每日最低工作时长

## 📊 班次类型说明

| 班次类型 | 说明 | 适用场景 |
|----------|------|----------|
| 标准班 | 固定上下班时间 | 行政岗位 |
| 早班 | 早间工作时段 | 生产一线 |
| 中班 | 下午工作时段 | 生产一线 |
| 晚班 | 晚间工作时段 | 生产一线 |
| 弹性班 | 灵活上下班时间 | 研发岗位 |
| 通宵班 | 跨日工作时段 | 值班岗位 |

---

## 🔧 配置参数

| 参数 | 类型 | 说明 |
|------|------|------|
| shift_name | String | 班次名称 |
| shift_code | String | 班次代码 |
| work_start_time | Time | 上班时间 |
| work_end_time | Time | 下班时间 |
| break_start_time | Time | 休息开始时间 |
| break_end_time | Time | 休息结束时间 |
| early_clock_minutes | Integer | 提前打卡分钟数 |
| late_clock_minutes | Integer | 延后打卡分钟数 |
| flex_start_time | Time | 弹性开始时间 |
| flex_end_time | Time | 弹性结束时间 |
| min_work_hours | Decimal | 最低工作时长 |
