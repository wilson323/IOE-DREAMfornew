# 餐别分类模块

> **版本**: v2.0.0  
> **更新日期**: 2025-12-17

---

## 1. 功能说明

两级餐别分类体系：餐别分类→具体餐别。

### 核心功能
- 餐别分类管理（早餐类/午餐类/晚餐类等）
- 具体餐别配置（时间段、基础价格）
- 账户类别餐别权限
- 区域餐别关联

---

## 2. 用户故事

| 角色 | 故事 |
|------|------|
| 管理员 | 创建餐别分类和具体餐别 |
| 管理员 | 配置餐别时间段和价格 |
| 管理员 | 为账户类别配置可用餐别 |

---

## 3. 数据结构

### POSID_MEAL_CATEGORY 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(50) | 分类ID |
| code | VARCHAR(50) | 分类编号 |
| name | VARCHAR(100) | 分类名称 |

### POSID_MEAL 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | VARCHAR(50) | 餐别ID |
| category_id | VARCHAR(50) | 所属分类 |
| start_time | VARCHAR(10) | 开始时间 |
| end_time | VARCHAR(10) | 结束时间 |
| base_price | INT | 基础价格(分) |

---

## 4. 业务流程

```
创建分类 --> 创建餐别 --> 配置时间段 --> 配置价格 --> 关联区域 --> 启用
```
