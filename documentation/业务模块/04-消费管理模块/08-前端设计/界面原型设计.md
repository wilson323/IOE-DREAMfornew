## 🎨 组件设计

### 消费统计卡片组件

```vue
<template>
  <el-card class="stat-card">
    <div class="stat-icon" :style="{ background: color }">
      <el-icon><component :is="icon" /></el-icon>
    </div>
    <div class="stat-content">
      <div class="stat-value">{{ value }}</div>
      <div class="stat-label">{{ label }}</div>
    </div>
    <div class="stat-trend" :class="{ up: trend > 0, down: trend < 0 }">
      {{ trend > 0 ? '+' : '' }}{{ trend }}%
    </div>
  </el-card>
</template>
```

### 消费记录表格组件

```vue
<template>
  <el-table :data="records" border stripe>
    <el-table-column prop="tradeNo" label="流水号" width="150" />
    <el-table-column prop="userName" label="用户" width="100" />
    <el-table-column prop="areaName" label="区域" width="120" />
    <el-table-column prop="amount" label="金额" width="100">
      <template #default="{ row }">
        ¥{{ row.amount.toFixed(2) }}
      </template>
    </el-table-column>
    <el-table-column prop="consumeType" label="方式" width="80">
      <template #default="{ row }">
        <el-tag :type="getTypeTag(row.consumeType)">
          {{ getTypeName(row.consumeType) }}
        </el-tag>
      </template>
    </el-table-column>
    <el-table-column prop="consumeTime" label="时间" width="160" />
    <el-table-column label="操作" width="100" fixed="right">
      <template #default="{ row }">
        <el-button type="primary" link @click="showDetail(row)">
          详情
        </el-button>
      </template>
    </el-table-column>
  </el-table>
</template>
```

## 📱 响应式设计

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| xs | < 576px | 单列布局，隐藏非必要列 |
| sm | ≥ 576px | 双列卡片 |
| md | ≥ 768px | 三列卡片 |
| lg | ≥ 992px | 四列卡片 |
| xl | ≥ 1200px | 完整布局 |
