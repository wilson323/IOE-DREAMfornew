## 📋 模块概述

设备通讯模块是IOE-DREAM智慧园区系统的核心基础设施，负责与门禁、考勤、消费等终端设备进行通讯，实现设备数据的采集、解析、转发和命令下发。

### 核心能力

- **多协议支持**: 支持门禁、考勤、消费等多种PUSH协议
- **组件化设计**: 协议处理器采用策略模式，易于扩展
- **高性能**: 基于Netty实现高并发设备连接
- **消息队列**: 使用RabbitMQ实现异步消息处理
