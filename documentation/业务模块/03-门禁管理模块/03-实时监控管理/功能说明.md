# 实时监控管理 - 功能说明

> **版本**: v1.0.0  
> **微服务**: ioedream-access-service (8090)  
> **创建日期**: 2025-12-17

---

## 📋 功能概述

实时监控管理提供门禁系统的实时状态监控、告警处理、视频联动和人员追踪功能。

---

## 🎯 核心功能

### 1. 设备状态实时监控
- **功能**: 监控所有门禁设备的在线/离线状态
- **更新频率**: ≤ 5秒
- **展示方式**: 大屏监控、列表视图、地图视图

### 2. 通行事件实时推送
- **功能**: WebSocket推送实时通行事件
- **事件类型**: 正常通行、异常通行、设备告警
- **关联代码**: `MonitorAlertService.java`

### 3. 告警处理
- **功能**: 异常事件自动告警、人工处理闭环
- **告警级别**: 低、中、高、紧急
- **响应时间**: 紧急告警 ≤ 30秒响应

### 4. 视频联动
- **功能**: 告警事件自动关联视频画面
- **联动方式**: 调用视频服务获取实时流
- **关联代码**: `VideoLinkageMonitorService.java`

### 5. 人员追踪
- **功能**: 追踪人员在各门禁点的通行轨迹
- **定位精度**: ≤ 2米
- **关联代码**: `AdvancedAccessControlService.java`
