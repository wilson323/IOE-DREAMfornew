## 🏗️ 系统架构

```mermaid
graph TB
    subgraph 配置中心
        Nacos[Nacos Config]
    end
    
    subgraph 门禁服务
        CFG[配置模块]
        BKP[备份模块]
    end
    
    subgraph 存储
        MySQL[(MySQL)]
        OSS[(OSS)]
    end
    
    Nacos --> CFG
    CFG --> MySQL
    BKP --> MySQL
    BKP --> OSS
```

---

## 📦 模块设计

| 模块 | 职责 |
|------|------|
| 配置服务 | 参数CRUD、变更通知 |
| 备份服务 | 数据备份恢复 |
| 审计服务 | 操作日志记录 |

---

## 🔧 技术选型

| 技术 | 选型 | 理由 |
|------|------|------|
| 配置中心 | Nacos | 动态配置 |
| 文件存储 | MinIO/OSS | 备份存储 |
| 压缩算法 | GZIP | 标准压缩 |
