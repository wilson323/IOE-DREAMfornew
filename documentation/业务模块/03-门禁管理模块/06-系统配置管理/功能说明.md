# 系统配置管理 - 功能说明

> **版本**: v1.0.0  
> **微服务**: ioedream-access-service (8090)  
> **创建日期**: 2025-12-17

---

## 📋 功能概述

系统配置管理提供门禁系统的参数配置、用户权限管理、许可证管理和备份恢复功能。

---

## 🎯 核心功能

### 1. 系统参数配置
- **功能**: 配置门禁系统全局参数
- **内容**: 心跳间隔、超时时间、认证阈值等
- **配置来源**: Nacos配置中心

### 2. 用户权限管理
- **功能**: 管理系统用户和角色权限
- **集成**: Sa-Token权限框架
- **层级**: 功能权限、数据权限、操作权限

### 3. 安全配置
- **功能**: 配置安全策略和审计规则
- **内容**: 密码策略、登录限制、审计开关

### 4. 备份恢复
- **功能**: 配置数据的备份和恢复
- **范围**: 系统配置、规则数据、模板数据

---

## 📊 配置项列表

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 心跳间隔 | 30秒 | 设备心跳检测间隔 |
| 离线阈值 | 5分钟 | 判定设备离线的时间 |
| 认证超时 | 10秒 | 认证请求超时时间 |
| 日志保留 | 365天 | 通行日志保留天数 |
