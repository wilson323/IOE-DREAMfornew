> **版本**: v2.0.0  
> **更新日期**: 2025-12-17  
> **适用范围**: IOE-DREAM智慧园区门禁管理系统

---

## 📋 模块概述

门禁管理模块是IOE-DREAM智慧园区系统的核心安全模块，基于ZKBioSecurity-ACC门禁系统架构，提供完整的门禁设备管理、区域权限控制、实时监控、事件记录、审批流程和高级安全功能。

### 系统架构

- **微服务**: ioedream-access-service (端口: 8090)
- **技术栈**: Spring Boot 3.5.8 + Java 17
- **数据库**: MySQL 8.0 + Druid连接池
- **缓存**: Redis + Caffeine多级缓存
- **认证**: Sa-Token

## 🎯 核心功能模块

### 1. 设备管理模块
- 设备管理：添加、编辑、删除、查询门禁设备
- 门管理：门参数配置、开门时间、验证方式
- 读头管理：读头参数配置、验证模式
- 设备控制：远程开锁、重启、同步数据
- 设备监控：在线状态、故障告警

### 2. 区域权限管理模块
- 区域信息管理：区域查询、修改、统计
- 门管理：区域内门的创建、配置
- 人员管理：区域人员分配、权限配置
- 门禁设备权限：设备验证权限独立建模，避免与软件权限混用
- 权限自动分配：新增门/人员自动分配权限

### 3. 实时监控模块
- 状态监控：实时设备状态、门状态监控
- 报警处理：报警接收、分析、推送、处理
- 视频联动：门禁事件触发视频录制
- 人员追踪：人员轨迹追踪和分析

### 4. 事件记录查询模块
- 门禁打卡事件：通行验证记录查询
- 报警异常事件：异常事件查询和处理
- 统计报表：多维度统计分析

### 5. 审批流程管理模块
- 权限申请：权限申请和审批流程
- 访客预约：访客预约和审批
- 紧急权限：紧急权限快速审批
- 流程配置：自定义审批流程

### 6. 系统配置模块
- 系统参数配置
- 用户权限管理
- 许可证管理
- 备份恢复

### 7. 高级功能模块
- 全局反潜：防止尾随和重复验证
- 全局联动：设备间联动响应
- 全局互锁：门之间的互锁控制
- 疏散点管理：紧急疏散管理
- 人数控制：区域容量控制
- 人员限制：访问限制规则

### 8. 多因子认证模块
- 人脸识别
- 指纹识别
- 掌纹识别
- 虹膜识别
- 声纹识别
- 动态安全策略
