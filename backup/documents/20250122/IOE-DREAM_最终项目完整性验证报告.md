   - MySQL 8.0集群

   - 对象存储（MinIO）

2. **部署顺序**
   ```bash
   # 1. 基础设施服务
   kubectl apply -f deployment/kubernetes/

   # 2. 数据库初始化
   mysql -u root -p < database-scripts/init.sql

   # 3. 微服务部署
   docker-compose -f docker-compose-production.yml up -d

   # 4. 前端部署
   npm run build:prod
   # 部署到CDN或静态服务器
   ```
