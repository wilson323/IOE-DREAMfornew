### 1. 开发者快速开始

#### 🔧 使用Swagger UI
```bash
# 本地环境
http://localhost:8080/swagger-ui.html

# 在线测试环境
https://test.ioe-dream.com/swagger-ui.html
```

#### 📋 导入Postman
1. 下载 `POSTMAN_COLLECTION.json`
2. 在Postman中导入
3. 配置环境变量
4. 开始测试

### 2. 测试人员使用指南

#### 🧪 自动化测试
```bash
# 运行完整测试套件
pytest tests/ --html=report.html

# 特定模块测试
pytest test_users.py --verbose

# 性能测试
python performance_monitor.py
```

#### 📊 生成测试报告
```bash
# HTML报告
pytest --html=reports/test_report.html

# 覆盖率报告
pytest --cov=tests --cov-report=html
```
