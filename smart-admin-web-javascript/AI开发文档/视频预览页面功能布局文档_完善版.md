# 视频预览页面功能与布局文档

## 页面概述
视频预览页面是智能视频监控系统的实时预览界面,支持多路视频实时播放、设备树选择、多画面布局切换(1/4/9/16画面)、云台控制、截图录像等核心功能。基于SmartAdmin框架,采用Vue3 + Ant Design Vue技术栈,集成jessibuca/video.js播放器,支持WebRTC/FLV/HLS多协议流媒体播放。

## 技术架构规范

### 前端技术栈
- **核心框架**: Vue 3.4.27 (Composition API)
- **UI组件库**: Ant Design Vue 4.2.5
- **状态管理**: Pinia 2.1.7
- **路由管理**: Vue Router 4.3.2
- **HTTP客户端**: Axios 1.6.8
- **图标库**: @ant-design/icons-vue
- **视频播放器**: jessibuca 3.x / video.js 8.x

### 视频播放器选型

#### 方案一: Jessibuca (推荐)
**优势**: 低延迟(200-500ms)、支持WebSocket-FLV/WebRTC、专为监控设计、支持H.264/H.265
```bash
npm install jessibuca
```

#### 方案二: Video.js
**优势**: 成熟稳定、插件丰富、支持HLS/DASH标准协议
```bash
npm install video.js
npm install @videojs/http-streaming
```

### API接口设计
```javascript
// src/api/business/smart-video/video-preview-api.js
import { postRequest, getRequest } from '/@/lib/axios';

export const videoPreviewApi = {
  // 获取设备树
  getDeviceTree: (params) => getRequest('/video/device/tree', params),

  // 获取视频流地址
  getStreamUrl: (deviceId, channelId) =>
    getRequest(`/video/stream/url/${deviceId}/${channelId}`),

  // 开始播放
  startPlay: (params) => postRequest('/video/play/start', params),

  // 停止播放
  stopPlay: (params) => postRequest('/video/play/stop', params),

  // 云台控制
  ptzControl: (params) => postRequest('/video/ptz/control', params),

  // 截图
  captureSnapshot: (params) => postRequest('/video/capture', params),

  // 开始录像
  startRecord: (params) => postRequest('/video/record/start', params),

  // 停止录像
  stopRecord: (params) => postRequest('/video/record/stop', params),

  // 获取预置位
  getPresetList: (deviceId) => getRequest(`/video/ptz/preset/${deviceId}`),

  // 调用预置位
  gotoPreset: (params) => postRequest('/video/ptz/preset/goto', params),

  // 设备统计
  getDeviceStats: () => getRequest('/video/device/statistics'),
};
```

### 3. 播放器内存管理

#### ✅ 正确的生命周期管理
```javascript
import { ref, onMounted, onUnmounted } from 'vue';

const playerInstances = ref([]);

onMounted(() => {
  initPlayers();
});

onUnmounted(() => {
  // 销毁所有播放器
  playerInstances.value.forEach(player => {
    if (player) {
      player.pause();
      player.destroy();
    }
  });
  playerInstances.value = [];
});

// 切换布局时清理
const changeLayout = (layout) => {
  playerInstances.value.forEach(p => p?.destroy());
  playerInstances.value = [];
  currentLayout.value = layout;
  nextTick(() => initPlayers());
};
```

### 4. 视频流协议支持
```javascript
const STREAM_PROTOCOL = {
  HLS: 'hls',          // HTTP Live Streaming
  FLV: 'flv',          // Flash Video
  WS_FLV: 'ws-flv',    // WebSocket FLV
  WEBRTC: 'webrtc',    // WebRTC
};

const getStreamUrl = (device, protocol) => {
  const { streamBaseUrl, channelId } = device;
  const urls = {
    hls: `${streamBaseUrl}/live/${channelId}.m3u8`,
    flv: `${streamBaseUrl}/live/${channelId}.flv`,
    'ws-flv': `ws://${streamBaseUrl}/live/${channelId}.flv`,
    webrtc: `${streamBaseUrl}/webrtc/${channelId}`,
  };
  return urls[protocol] || streamBaseUrl;
};
```

<script setup name="VideoPreview">
import { ref, reactive, onMounted, onUnmounted, nextTick } from 'vue';
import { message } from 'ant-design-vue';
import {
  VideoCameraOutlined,
  CameraOutlined,
  ExpandOutlined,
  FullscreenOutlined,
  SoundOutlined,
  ReloadOutlined,
  ZoomInOutlined,
  ZoomOutOutlined,
  ArrowUpOutlined,
  ArrowDownOutlined,
  ArrowLeftOutlined,
  ArrowRightOutlined,
  HomeOutlined,
} from '@ant-design/icons-vue';
import { videoPreviewApi } from './api/video-preview-api';
import Jessibuca from 'jessibuca';

// ==================== 状态管理 ====================
const currentLayout = ref(4);
const videoQuality = ref('medium');
const isRecording = ref(false);
const showPtzControl = ref(false);
const selectedPreset = ref(null);

const deviceTree = ref([]);
const deviceStats = reactive({ online: 0, total: 0 });
const videoPlayers = ref([]);
const playerInstances = ref([]);
const presetList = ref([]);

// ==================== 生命周期 ====================
onMounted(() => {
  loadDeviceTree();
  initVideoGrid();
});

onUnmounted(() => {
  destroyAllPlayers();
});

// ==================== 设备树操作 ====================
const loadDeviceTree = async () => {
  try {
    const { data } = await videoPreviewApi.getDeviceTree();
    deviceTree.value = data.tree;
    deviceStats.online = data.statistics.online;
    deviceStats.total = data.statistics.total;
  } catch (error) {
    message.error('加载设备树失败');
  }
};

const refreshDeviceTree = () => {
  loadDeviceTree();
  message.success('设备列表已刷新');
};

const handleDeviceSelect = async (selectedKeys, { node }) => {
  if (node.deviceType !== 'camera') return;

  const emptyIndex = videoPlayers.value.findIndex(p => !p.deviceId);
  if (emptyIndex === -1) {
    message.warning('所有画面已占用，请先关闭一个');
    return;
  }

  try {
    const { data } = await videoPreviewApi.getStreamUrl(node.id, node.channelId);
    playVideo(emptyIndex, node, data.streamUrl);
  } catch (error) {
    message.error('获取视频流失败');
  }
};

// ==================== 视频播放 ====================
const initVideoGrid = () => {
  const count = currentLayout.value;
  videoPlayers.value = Array(count).fill(null).map(() => ({
    deviceId: null,
    deviceName: null,
    streamUrl: null,
  }));
};

const playVideo = (index, device, streamUrl) => {
  const containerId = `video-${index}`;

  // 销毁旧播放器
  if (playerInstances.value[index]) {
    playerInstances.value[index].destroy();
  }

  // 创建新播放器 (Jessibuca)
  const player = new Jessibuca({
    container: document.getElementById(containerId),
    videoBuffer: 0.2,
    isResize: true,
    loadingText: '加载中...',
    debug: false,
    showBandwidth: true,
    operateBtns: {
      fullscreen: false,
      screenshot: false,
      play: true,
      audio: true,
    },
  });

  player.play(streamUrl);

  playerInstances.value[index] = player;
  videoPlayers.value[index] = {
    deviceId: device.id,
    deviceName: device.name,
    streamUrl,
  };

  message.success(`已加载 ${device.name}`);
};

const destroyAllPlayers = () => {
  playerInstances.value.forEach(player => {
    if (player) {
      player.pause();
      player.destroy();
    }
  });
  playerInstances.value = [];
};

// ==================== 布局切换 ====================
const handleLayoutChange = () => {
  destroyAllPlayers();
  initVideoGrid();
  nextTick(() => {
    message.info(`已切换到 ${currentLayout.value} 画面布局`);
  });
};

// ==================== 工具栏操作 ====================
const toggleRecord = async () => {
  try {
    if (isRecording.value) {
      await videoPreviewApi.stopRecord({ players: videoPlayers.value });
      message.success('录制已停止');
    } else {
      await videoPreviewApi.startRecord({ players: videoPlayers.value });
      message.success('开始录制');
    }
    isRecording.value = !isRecording.value;
  } catch (error) {
    message.error('录制操作失败');
  }
};

const handleScreenshot = async () => {
  try {
    await videoPreviewApi.captureSnapshot({ players: videoPlayers.value });
    message.success('截图已保存');
  } catch (error) {
    message.error('截图失败');
  }
};

const toggleFullscreen = () => {
  const elem = document.querySelector('.video-grid-card');
  if (!document.fullscreenElement) {
    elem.requestFullscreen();
  } else {
    document.exitFullscreen();
  }
};

const toggleSound = () => {
  playerInstances.value.forEach(player => {
    if (player) {
      player.setMute(!player.isMute());
    }
  });
};

// ==================== PTZ 控制 ====================
const openPtzControl = async (index) => {
  const player = videoPlayers.value[index];
  if (!player.deviceId) {
    message.warning('请先选择设备');
    return;
  }

  try {
    const { data } = await videoPreviewApi.getPresetList(player.deviceId);
    presetList.value = data;
    showPtzControl.value = true;
  } catch (error) {
    message.error('加载预置位失败');
  }
};

const ptzControl = async (direction) => {
  try {
    await videoPreviewApi.ptzControl({ direction });
    message.success(`PTZ ${direction}`);
  } catch (error) {
    message.error('PTZ控制失败');
  }
};

const zoomControl = async (action) => {
  try {
    await videoPreviewApi.ptzControl({ action: `zoom_${action}` });
    message.success(`变焦 ${action === 'in' ? '放大' : '缩小'}`);
  } catch (error) {
    message.error('变焦失败');
  }
};

const gotoPreset = async () => {
  if (!selectedPreset.value) {
    message.warning('请选择预置位');
    return;
  }

  try {
    await videoPreviewApi.gotoPreset({ presetId: selectedPreset.value });
    message.success('已跳转到预置位');
  } catch (error) {
    message.error('跳转失败');
  }
};
</script>

### 核心功能说明
1. **设备树**: 左侧树形结构,支持分组管理,实时显示设备在线状态
2. **多画面**: 支持1/4/9/16画面自动布局,动态创建/销毁播放器
3. **视频播放**: 集成jessibuca,支持低延迟流媒体播放
4. **云台控制**: 方向控制、变焦、预置位调用
5. **录制截图**: 支持多路同时录制和截图
6. **全屏模式**: 支持全屏查看视频画面
