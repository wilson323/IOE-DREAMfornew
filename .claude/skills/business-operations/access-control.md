# 🚪 门禁管理技能

**技能名称**: 智能门禁系统操作与管理
**技能等级**: 中级
**适用角色**: 门禁管理员、安保人员、行政管理人员
**前置技能**: 人员管理基础、设备操作基础、计算机基础操作
**预计学时**: 12小时

---

## 📚 知识要求

### 📖 理论知识
- **门禁系统原理**: 理解门禁系统的基本组成和工作原理
- **识别技术**: 掌握IC卡、指纹、人脸、二维码等识别技术
- **权限管理**: 了解基于时间、区域、角色的权限控制机制
- **安防标准**: 了解门禁系统的安防标准和规范要求

### 💼 业务理解
- **门禁流程**: 掌握人员进出管理的完整业务流程
- **应急预案**: 熟悉门禁系统故障时的应急处理流程
- **访客管理**: 理解访客预约、登记、授权的业务流程
- **审计要求**: 了解门禁记录的审计和合规要求

### 🔧 技术背景
- **设备操作**: 熟悉门禁控制器、读卡器、电锁等设备操作
- **系统界面**: 掌握门禁管理软件的操作界面和功能
- **数据处理**: 了解门禁数据的查询、统计和导出方法
- **网络基础**: 基本的网络连接和故障判断能力

## 🛠️ 操作步骤

### 步骤1: 系统登录和界面熟悉 (1小时)

#### 1.1 系统登录
```
访问地址: http://localhost:8081
用户名: [分配的用户名]
密码: [分配的密码]

1. 打开浏览器，输入系统地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 验证登录成功，进入系统主页
```

#### 1.2 主界面功能熟悉
- **导航菜单**: 门禁管理、人员管理、设备管理、记录查询
- **快捷操作**: 常用功能快速访问
- **状态显示**: 系统状态、设备在线情况
- **消息通知**: 系统公告、异常提醒

### 步骤2: 人员权限管理 (3小时)

#### 2.1 新增人员门禁权限
```
操作路径: 门禁管理 → 人员权限 → 新增权限

1. 选择要授权的人员
   - 支持按姓名、工号搜索
   - 支持按部门批量选择

2. 配置权限信息:
   - 有效期限: 设置权限生效时间
   - 通行区域: 选择允许通行的门禁点
   - 通行时段: 设置允许通行的时间段
   - 权限级别: 普通用户/管理员/超级管理员

3. 下发权限到设备:
   - 选择目标门禁设备
   - 点击"下发权限"按钮
   - 确认下发结果
```

#### 2.2 批量人员权限配置
```
场景: 新员工入职批量授权

1. 人员信息导入:
   - 下载人员信息模板
   - 填写人员基本信息
   - 上传人员信息文件

2. 权限模板设置:
   - 创建权限模板
   - 配置通用权限参数
   - 保存权限模板

3. 批量应用权限:
   - 选择导入的人员
   - 应用权限模板
   - 批量下发到设备

4. 验证权限生效:
   - 查看下发状态
   - 测试门禁通行
   - 确认权限正确
```

### 步骤3: 设备管理操作 (2小时)

#### 3.1 门禁设备配置
```
操作路径: 设备管理 → 门禁设备 → 设备配置

1. 设备基础配置:
   - 设备名称: 如"正门门禁"
   - 设备位置: 具体安装位置
   - 设备编号: 唯一设备标识
   - 连接方式: TCP/IP/串口

2. 通行参数设置:
   - 开门延时时间: 3-10秒
   - 门磁检测: 启用/禁用
   - 尾随检测: 启用/禁用
   - 反潜功能: 启用/禁用

3. 报警参数配置:
   - 强行开门报警
   - 门未关报警
   - 胁迫报警
   - 通讯中断报警
```

#### 3.2 设备状态监控
```
操作路径: 设备管理 → 设备监控

1. 实时状态查看:
   - 设备在线状态: 绿色表示在线，红色表示离线
   - 通讯状态: 显示最后一次通讯时间
   - 设备版本: 显示固件版本信息
   - 运行状态: 正常/异常/维护中

2. 异常处理:
   - 设备离线: 检查网络连接和电源
   - 通讯异常: 重启设备或检查网络配置
   - 读写器故障: 检查硬件连接或更换设备
   - 电控锁故障: 检查电源或机械部分
```

### 步骤4: 门禁记录管理 (2小时)

#### 4.1 实时记录监控
```
操作路径: 记录查询 → 实时监控

1. 选择监控范围:
   - 监控设备: 选择要监控的门禁点
   - 显示字段: 人员、时间、事件类型、结果
   - 自动刷新: 设置刷新间隔（5-60秒）

2. 异常事件处理:
   - 识别异常记录（红色显示）
   - 点击记录查看详细信息
   - 根据情况采取相应措施
   - 记录处理结果和备注
```

#### 4.2 历史记录查询
```
操作路径: 记录查询 → 历史查询

1. 设置查询条件:
   - 时间范围: 开始时间-结束时间
   - 人员范围: 指定人员或部门
   - 设备范围: 指定门禁点
   - 事件类型: 正常通行/异常事件

2. 数据导出:
   - 选择导出格式: Excel/PDF/CSV
   - 设置导出字段
   - 点击"导出"按钮
   - 保存导出文件

3. 统计分析:
   - 通行次数统计
   - 异常事件统计
   - 人员活跃度分析
   - 设备使用率统计
```

### 步骤5: 访客管理操作 (2小时)

#### 5.1 访客预约和登记
```
操作路径: 访客管理 → 访客预约

1. 访客预约:
   - 被访人信息: 选择被访员工
   - 访客信息: 姓名、身份证、联系电话
   - 访问事由: 详细说明访问目的
   - 访问时间: 预计到达和离开时间
   - 通行区域: 允许通行的区域

2. 访客登记:
   - 身份证件验证
   - 拍照存档
   - 发放访客卡/二维码
   - 权限即时生效
```

#### 5.2 访客权限管理
```
操作路径: 访客管理 → 权限管理

1. 权限下发:
   - 选择访客记录
   - 设置通行权限
   - 选择有效时间
   - 下发到相关设备

2. 权限回收:
   - 访客离开时及时回收权限
   - 访客卡回收或二维码失效
   - 记录离开时间
   - 生成访客报告
```

### 步骤6: 应急处理流程 (2小时)

#### 6.1 系统故障应急处理
```
场景: 门禁系统突然无法正常工作

1. 快速诊断:
   - 检查网络连接
   - 检查设备电源
   - 查看系统日志
   - 联系技术支持

2. 应急方案:
   - 手动开门授权
   - 临时通行证发放
   - 安保人员现场值守
   - 人工登记记录

3. 恢复操作:
   - 系统功能恢复
   - 补录通行记录
   - 权限验证测试
   - 恢复正常管理
```

#### 6.2 紧急事件处理
```
场景: 火警、安防等紧急情况

1. 紧急开门:
   - 激活紧急开门模式
   - 所有门禁自动解锁
   - 释放电磁锁
   - 启动疏散指示

2. 事件记录:
   - 自动记录紧急事件
   - 保存操作日志
   - 生成事件报告
   - 通知相关人员

3. 恢复正常:
   - 解除紧急状态
   - 恢复正常权限
   - 检查设备状态
   - 确认系统正常
```

## ⚠️ 注意事项

### 🔒 安全提醒
- **权限最小化**: 严格按照最小权限原则分配门禁权限
- **定期审核**: 定期审核人员权限，及时清理过期权限
- **密码安全**: 系统登录密码定期更换，不得转借他人
- **应急备用**: 保留机械钥匙等备用开门方式

### 📊 质量要求
- **权限准确率**: 人员权限配置准确率必须达到100%
- **响应时间**: 权限下发响应时间应小于10秒
- **记录完整性**: 门禁记录完整率应达到99.9%以上
- **系统可用性**: 门禁系统可用性应达到99.5%以上

### 🎯 最佳实践
- **权限分级**: 根据人员级别和工作需要分级授权
- **时段管理**: 合理设置通行时段，避免非工作时间通行
- **区域控制**: 严格按照工作区域划分通行权限
- **定期检查**: 定期检查设备运行状态和权限配置
