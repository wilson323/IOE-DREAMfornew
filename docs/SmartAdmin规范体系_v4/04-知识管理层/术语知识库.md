# 术语知识库（权威文档）

> **📋 文档版本**: v4.0.0 (整合版)
> **📋 文档职责**: SmartAdmin项目的统一术语定义，确保团队沟通的一致性和准确性。

## 📚 术语分类

### 架构相关术语

#### 四层架构
- **Controller层**: 接口控制层，负责接收HTTP请求、参数验证、响应封装
- **Service层**: 业务逻辑层，负责核心业务逻辑处理、事务管理
- **Manager层**: 复杂业务层，负责复杂业务逻辑、缓存管理、第三方集成
- **Repository层**: 数据访问层，负责数据库CRUD操作、SQL执行

#### 依赖注入
- **@Resource**: 依赖注入注解，必须使用的注入方式
- **@Autowired**: 被禁止使用的注入注解
- **IoC**: 控制反转，依赖倒置原则的应用

### 数据相关术语

#### 实体类命名
- **Entity**: 数据库实体类，如UserEntity
- **Form**: 表单类，如UserAddForm、UserUpdateForm
- **VO**: 视图对象类，如UserVO
- **DTO**: 数据传输对象（不推荐使用）

#### 数据库命名
- **表名**: t_{business}_{entity}格式，如t_user_info
- **主键**: {entity}Id格式，如userId
- **审计字段**: create_time、update_time、deleted_flag、version

### 业务相关术语

#### 权限控制
- **RBAC**: 基于角色的访问控制
- **Sa-Token**: 权限控制框架
- **@SaCheckPermission**: 权限验证注解
- **@SaCheckLogin**: 登录验证注解

#### 状态管理
- **状态标记**: status字段，1-正常，0-禁用
- **软删除**: deleted_flag字段，0-正常，1-删除
- **乐观锁**: version字段，用于并发控制

### 技术相关术语

#### 缓存技术
- **L1缓存**: Caffeine本地缓存
- **L2缓存**: Redis分布式缓存
- **Cache Aside**: 缓存更新策略
- **缓存穿透**: 查询不存在数据绕过缓存

#### 消息队列
- **RocketMQ**: 消息队列中间件
- **Producer**: 消息生产者
- **Consumer**: 消息消费者
- **Topic**: 消息主题

### 安全相关术语

#### 认证授权
- **JWT**: JSON Web Token
- **Token**: 访问令牌
- **Session**: 会话管理
- **BCrypt**: 密码加密算法

#### 安全防护
- **SQL注入**: SQL Injection攻击
- **XSS**: 跨站脚本攻击
- **CSRF**: 跨站请求伪造
- **OWASP**: Web安全防护标准

### 性能相关术语

#### 响应时间
- **P95**: 95%请求的响应时间
- **P99**: 99%请求的响应时间
- **QPS**: 每秒查询率
- **TPS**: 每秒事务处理量

#### 数据库性能
- **索引**: 数据库索引优化
- **分页**: 数据分页查询
- **读写分离**: 主从数据库分离
- **分库分表**: 大数据量分表策略

### 开发相关术语

#### 代码质量
- **圈复杂度**: 代码复杂度度量
- **代码覆盖率**: 单元测试覆盖率
- **代码重复率**: 重复代码占比
- **SonarQube**: 代码质量检测工具

#### 版本控制
- **Git**: 版本控制系统
- **Branch**: 分支管理
- **Merge**: 代码合并
- **Conflict**: 代码冲突

---

**📋 术语维护原则**：
1. **统一性** - 整个项目使用统一术语
2. **准确性** - 术语定义准确清晰
3. **简洁性** - 术语简洁易懂
4. **一致性** - 术语使用保持一致
